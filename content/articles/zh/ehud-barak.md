## 职业年表

| 年份 | 事件 |
|------|------|
| 1942 | 出生于英属巴勒斯坦托管地（现以色列）。[^1] |
| 1959 | 加入以色列国防军（IDF）。[^1] |
| 1991–1995 | 担任以色列国防军总参谋长（军衔：少将/中将）。[^1] |
| 1995–1996 | 在总理希蒙·佩雷斯领导下担任以色列外交部长。[^1] |
| 1999 | 当选以色列总理。[^1] |
| 2000 | 主导以色列从黎巴嫩南部撤军；参加戴维营峰会。[^1] |
| 2001 | 在总理特别选举中落败并辞去总理职务。[^1] |
| 2007–2013 | 担任国防部长（包括在总理本雅明·内塔尼亚胡任期内）。[^1] |
| 2011–2013 | 脱离工党并领导独立党，此后退出政坛。[^1] |

## 背景（概要）

- **军事生涯：** 巴拉克早年大部分时间在以色列国防军服役，后来成为总参谋长。[^1]
- **政治生涯：** 从以色列国防军退役后，他加入以色列工党从政，并担任过外交部长和国防部长等高级内阁职务。[^1]
- **总理任期（1999–2001）：** 他的任期通常与2000年代初期重大的地区外交努力和国内政治动荡联系在一起。[^1]
