## 概述

参见 阿布·贝克尔·巴格达迪。伊斯兰国武装组织。重复条目。

## 生平大事记

| 年份 | 事件 |
|------|-------|
| 2004 | 起源于伊拉克，由 阿布·穆萨布·扎卡维领导，通常追溯到 **统一与圣战组织 (Jama'at al-Tawhid wal-Jihad)**，后成为 **伊拉克基地组织 (AQI)**。 |
| 2006 | AQI 宣布成立一个伞状组织后，重新命名为 **伊拉克伊斯兰国 (ISI)**。 |
| 2013 | 扩张至叙利亚；媒体和政策中广泛使用 **伊拉克和黎凡特伊斯兰国 (ISIS/ISIL)** 的名称。 |
| 2014 | 占领摩苏尔，宣布成立由 **阿布·贝克尔·巴格达迪** 领导的“哈里发国”；在伊拉克和叙利亚拥有大片领土。 |
| 2015–2017 | 在多方军事行动下，领土损失加速；宣传和外部袭击成为主要关切。 |
| 2019 | 在叙利亚（巴古兹）失去最后主要领土；组织转向叛乱和分散的网络。 |
| 2019 | 巴格达迪在美军突袭中丧生；随后该组织宣布了新的领导人。 |

## 关于名称和范围的说明

- **ISIS / ISIL / IS** 通常指代同一组织在不同时期和不同命名约定下的名称。
- 此页面作为一个简短的标识符条目。主要传记资料请参阅 **阿布·贝克尔·巴格达迪** 的原始文件。

## 概览 (高层)

- **类型：** 逊尼派伊斯兰武装和恐怖组织。
- **核心主张（2014–2019 年高峰期）：** 建立“哈里发国”并统治被占领土。
- **行动：** 叛乱、领土控制（高峰期）、武装袭击以及广泛的媒体/宣传活动。
- **国际地位：** 被众多政府和国际机构广泛指定为恐怖组织。

## 意识形态和目标 (概述)

- 通常被描述为信奉对 **逊尼派伊斯兰教的极端、暴力解读**。
- 目标包括：
  - 推翻中东部分地区现有的国家结构。
  - 建立符合其意识形态计划的治理。
  - 通过宣传和社会媒体传播，在全球范围内招募支持者。

## 当前状态 (总体)

- 在 2019 年遭受重大领土失败后，据报道 ISIS 继续作为一个 **叛乱运动**，并在多个地区拥有 **附属或受其启发的网络**。
- 其规模、组织和作战能力随时间和地理位置而显著变化。
