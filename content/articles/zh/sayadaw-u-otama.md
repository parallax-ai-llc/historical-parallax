## 简介

赛亚道·吴·奥塔玛 (1879-1939) 是一位上座部佛教僧侣，他在英国殖民统治期间成为缅甸独立运动的先驱领袖。他被称为“缅甸的甘地”，是第一位踏入政坛的僧侣，也是英属缅甸第一位因发表政治演说而被监禁的人。受到甘地非暴力抵抗原则的启发，吴·奥塔玛走遍全国，建立了民族主义组织，提倡抵制英货，并动员僧侣和普通民众反对殖民压迫。在1921年至1927年间，他入狱的时间比出狱的时间还多，他不知疲倦的活动为缅甸最终于1948年独立奠定了基础。[^1][^2][^3]

---

## 早年生活与背景

吴·奥塔玛，原名吴貌吞（Paw Tun Aung），于1879年12月28日出生在缅甸西部若开邦（今实兑）附近的路巴村。[^4] 他是吴貌（U Mra）和吴貌（Daw Aung Kwa Pyu）的儿子，家境贫寒，生活拮据。[^5] 尽管出身卑微，奥塔玛和他的兄弟觉吞貌（Kyaw Tun Aung，后改名阿里亚 Ariya）都在英国人开办的乡村学校接受了基础教育，年轻的奥塔玛展现出快速的学习能力。[^6]

为了寻求更深入的宗教教育，年轻的貌吞貌乔装打扮前往邻近马圭省著名的帕科库寺院，并剃度出家。[^7] 奥塔玛在他村校学习期间就已展现出卓越的智力，在寺院里他成绩优异，获得了许多资深僧侣和认识到他潜力的经济资助者的赞助。[^8] 这些资助者帮助他申请加尔各答大学，为他未来成为宗教领袖和政治革命家铺平了道路。[^9]

十九岁时，他完成了入学考试，前往印度，在加尔各答大学学习了数年，主修佛教研究、巴利语和梵语。[^10] 多年来，奥塔玛在加尔各答大学担任巴利语和佛教文学讲师，展现了他出色的语言天赋，迅速成为一个语言大师，学会并教授梵语、英语、日语以及许多其他语言。[^11] 后来他回到缅甸，正式剃度出家，取名奥塔玛，在巴利语中意为“至高无上”或“崛起”。[^12] 他在帕科库活跃了一段时间，然后回到阿拉干的瑞泽迪寺院学习，在那里他帮助他的兄弟觉吞貌剃度出家。[^13]

## 崛起

吴·奥塔玛从学者型僧侣转变为革命领袖，始于他20世纪初的广泛国际旅行。在加尔各答学习三年后，他游历印度，接触到了蓬勃发展的印度独立运动。[^14] 在印度，他与印度国民大会党以及圣雄甘地关系密切，甘地的非暴力抵抗哲学对奥塔玛的反殖民斗争方法产生了深远影响。[^15] 他还访问了法国和埃及，拓宽了他的世界观，加深了他对反殖民运动的理解。[^16]

1907年1月，吴·奥塔玛前往日本，在东京佛教科学学院教授巴利语和梵语。[^17] 这使他成为第一批在海外学习和教学的缅甸僧侣之一。[^18] 随后，他广泛游历了朝鲜、满洲、旅顺、中国、安南、柬埔寨、泰国、斯里兰卡和印度，以及美国和英国。[^19] 这些旅行让他接触到各种独立运动和政治思想，他后来将这些带回缅甸，领导了反对英国统治的反殖民斗争。[^20]

回到英属缅甸后，吴·奥塔玛开始了他的政治活动，游历全国，为青年佛教徒协会（YMBA）发表演讲，发表反殖民言论，吸引了大批追随者。[^21] 他成为了YMBA的一员，为民族主义运动注入了新的活力，缅甸大部分有影响力的佛教僧侣（Pongyis）都受到了他的影响。[^22] 作为一名激情的演说家和鼓动者，吴·奥塔玛吸引了大量追随者，主要是佛教僧侣，他们组织了示威和集会，这让英国殖民当局非常警惕，他们动用军警驱散了这些集会。[^23]

吴·奥塔玛在他所到之处都创立了“Wunthanu Athins”（爱国社团或民族主义协会）。[^24] 到1924年，被称为“Wunthanu Athins”的组织几乎遍布缅甸的每个村庄，在反对英国殖民统治的政治动员中发挥了重要作用。[^25] 这些组织在地方层面运作，负责弘扬缅甸传统、民族认同和佛教的自豪感，并煽动反对1907年《乡村法》等压迫性措施。[^26] 吴·奥塔玛积极参与新兴的政治运动，为民族主义报纸《苏里亚》（The Sun）撰写文章，在村庄里布道，并建立“Wunthanu Athins”的分支机构。[^27]

## 主要成就

吴·奥塔玛最显著的成就是将佛教僧侣制度与政治活动相结合，开创了缅甸独立斗争的先河。奥塔玛是第一批踏入政坛的僧侣之一，也是英属缅甸第一位因发表政治演说而被监禁的人，之后还有安吉·桑和吴努等众多民族主义者。[^28] 许多人认为他是该国独立运动之父，并被视为缅甸民族主义的第一位真正烈士以及现代若开民族主义运动之父。[^29]

受到甘地主义原则的启发，吴·奥塔玛倡导非暴力抵抗，提倡和平抗议、抵制英货以及复兴本土价值观和自力更生。[^30] 以甘地的方式，他将一个基本的政治问题——缅甸的民族主义和独立——转化为一个宗教问题，吸引了包括未受过英国教育的人在内的所有人。[^31] 他通过佛教伦理来表达对殖民统治的反对，创造了一个强烈引起缅甸人民共鸣的动人信息。[^32] 这种将宗教认同融入政治活动的独特关注，确立了民族主义斗争与社区之间深厚的联系。[^33]

他最著名的反抗行为发生在1921年，当时他发表了臭名昭著的“克拉多克，滚出去！”（Craddock, Get Out!）演讲，反对时任英属缅甸总督雷金纳德·克拉多克爵士提出的克拉多克计划。[^34] 这次演讲导致他因煽动叛乱罪被捕入狱，成为英属缅甸第一位因发表政治演说而被监禁的人。[^35] 在当时缅甸许多人甚至害怕最低级的英国官员的情况下，吴·奥塔玛激昂的言论具有激励人们、消除在帝国主义面前的恐惧的力量，结果，全国各地的民族主义协会蓬勃发展，僧侣们也开始布道反殖民的演说。[^36]

吴·奥塔玛通过倡导非暴力和抵制，类似于甘地的策略，在改变缅甸民族主义运动方面发挥了关键作用，他的影响力促使到1924年，缅甸农村地区广泛建立了民族主义协会。[^37] 在奥塔玛的启发下，大量佛教僧侣参与了政治运动。[^38] 他强调不合作和抵制英国改革，而非起义，提倡和平的政治变革途径。[^39] 他通过演讲和在《太阳报》上的爱国文章激励了缅甸人民，《太阳报》支持他，并在没有寺院接纳他作为政治僧侣时为他提供了媒体平台。[^40]

他非凡的贡献为他赢得了“亚洲之阳”的称号，他的爱国热情和勇气激励了最终于1948年赢得独立的年轻一代缅甸人。[^41] 1929年，他代表印度国民大会党参加了孙中山先生的葬礼，显示了他泛亚联系和国际上对他领导地位的认可。[^42]

## 争议与批评

尽管作为民族主义英雄备受尊敬，吴·奥塔玛的政治活动并非没有争议，特别是关于佛教僧侣在政治中的作用。“僧侣的政治煽动‘并没有得到广大民众的认可，因为这种公开参与反殖民政治或社会活动被视为违反了僧侣戒律。’”[^43] 缅甸僧伽的大多数资深僧侣反对僧侣参与政治，认为这种活动与约束僧侣行为的戒律相悖。[^44]

学者迈克尔·门德尔森写道，所有积极参与政治的僧侣往往被殖民当局贴上“黄色僧袍中的政治煽动者”的标签，这种定性试图使其宗教权威合法性受到质疑。[^45] 英国殖民政府视吴·奥塔玛为危险的激进分子，并以煽动叛乱罪多次监禁他。[^46] 据学者称，在1921年至1927年间，吴·奥塔玛入狱的时间比出狱的时间还多，忍受着恶劣的条件，这些条件最终损害了他的健康。[^47]

吴·奥塔玛的哲学方法也存在矛盾。虽然吴·奥塔玛忠于他的甘地主义理想，并且个人不提倡在反殖民斗争中使用暴力，但他认为英国统治是佛教实践的障碍，因此必须获得独立，如果必要，可以通过暴力手段。[^48] 为了支持在某些情况下的使用武力，他引用了一些“本生谭”（Jatakas，佛陀前世的故事），在他宣称的非暴力与反殖民抵抗的现实之间造成了紧张关系。[^49] 尽管他个人提倡非暴力，但在20世纪20年代，发生了几起涉及激进僧侣煽动反对殖民统治的暴力事件，其中一些声称受到了吴·奥塔玛教导的启发。[^50]

在他生命的最后几年，吴·奥塔玛遭受疾病、贫困和年轻一代缅甸民族主义者的忽视，他们已经超越了他的方法。[^51] 一些历史学家指出，虽然吴·奥塔玛在20世纪20年代初肯定影响了萨亚·桑，但这两位领导人之间的差异比相似之处更为显著。[^52] 吴·奥塔玛被描述为一位“世界主义的革命家”，而萨亚·桑的努力“不那么现代但更成功”，通过传统和宗教诉求而非现代民族主义计划动员了乡村农民。[^53]

## 个人生活

吴·奥塔玛在其整个职业生涯中都保持着佛教僧侣的简朴生活，即使在他成为一位杰出的政治人物后，也仍然遵守独身和简朴的戒律。他出生在一个贫困的若开家庭，从未忘记自己卑微的出身，并一直致力于服务缅甸的普通民众。[^54]

他天生的语言天赋让他迅速成为一个语言大师，学会并教授梵语、英语、日语以及许多其他语言。[^55] 这种语言能力使他在作为一名教师和一位能与亚洲各地不同听众交流的政治组织者方面都发挥了重要作用。他的兄弟觉吞貌（Kyaw Tun Aung），出家法号阿里亚（Ariya），在瑞泽迪寺院与他一同出家，尽管历史记录对他们之间的关系或阿里亚的活动所知甚少。[^56]

吴·奥塔玛以其激情的演说和富有魅力的风范而闻名。作为一名激情的演说家和鼓动者，他吸引了大量追随者，主要是佛教僧侣，他们组织了示威和集会来支持他。[^57] 他的演讲对他听众产生了深远的情感影响，在英国殖民权力似乎不可战胜的时期，驱散了恐惧，激励人们采取行动。[^58]

在1921年至1927年间反复被监禁期间，吴·奥塔玛忍受了巨大的困苦。英国殖民当局无视他的非暴力方法，对他施加严酷对待，这导致了他的健康状况恶化。[^59] 在他生命的最后几年，他患有糖尿病以及与多年监禁、长期旅行和活动造成的身体损耗相关的其他疾病。[^60] 《太阳报》支持他，并在没有寺院接纳他作为政治僧侣时为他提供了平台，这表明即使在佛教界，他的活动也具有争议性。[^61]

## 遗产

吴·奥塔玛于1939年9月9日因糖尿病在仰光医院去世，享年60岁，此前他因多年监禁和健康状况不佳而饱受折磨。[^62] 他的去世标志着缅甸独立运动一个时代的结束，但他的影响将继续塑造国家数十年。

吴·奥塔玛的活动促成了一个激进的、反殖民的运动的出现，该运动传播了民族主义、佛教以及在其后期阶段的社会主义的结合。[^63] 这种政治和宗教信仰的混合，在殖民时期的缅甸大多数民族主义政客那里以不同程度的坚持，并且在国家于1948年独立后，这一点几乎没有改变。[^64] 他的工作为后来的民族主义领导人，如昂山和吴努，在缅甸最终走向1948年独立铺平了道路。[^65]

他于1939年9月9日的逝世日被定为每年纪念的“吴·奥塔玛日”，以纪念他的生平，并提醒当代的世代缅甸人民为反抗殖民压迫所付出的长期斗争。[^66] 他的忌日广受若开邦各地（包括皎漂、丹兑等城镇）的纪念，并在泰国、马来西亚、韩国以及其他拥有缅甸侨民社区的国家举行纪念活动。[^67] 他在仰光（今阿拉干邦）的旧寺院瑞泽迪（Shwezedi）仍然是政治和反政府活动的重要中心。[^68]

吴·奥塔玛被认为是现代缅甸的民族英雄之一，他在独立斗争中的贡献在官方历史和公众记忆中得到认可。[^69] 他既是一位宗教人物，也是一位政治革命家，他证明了僧侣可以在坚持其精神承诺的同时，积极参与反抗不公。[^70] 他的榜样激励了缅甸一代又一代政治活跃的僧侣，尽管僧侣是否应参与政治的问题至今仍存在争议。[^71]

## 历史评价

### 积极评价

学者和历史学家普遍认为吴·奥塔玛是一位具有变革意义的人物，他成功地将佛教宗教身份与政治民族主义相结合，开创了塑造缅甸独立运动数十年方法的先河。[^72] 他通过倡导非暴力和抵制，类似于甘地的策略，在改变缅甸民族主义运动方面发挥了关键作用，他的影响力促成了1924年缅甸农村地区广泛民族主义协会的形成。[^73]

学界普遍认为吴·奥塔玛是一位奠基性人物，他将宗教身份融入政治活动的独特关注，建立了民族主义斗争与社区之间深厚的联系。[^74] 在政治意识浓厚的环境中成长并在印度学习，塑造了吴·奥塔玛非暴力抵抗的方法，这让人想起甘地主义原则，他对印度民族主义运动的接触和佛教教义对他产生了深远影响。[^75]

人权观察和其他国际组织已将吴·奥塔玛视为缅甸佛教活动主义传统的先驱人物，并指出他是第一批站起来反对英国殖民统治的僧侣。[^76] 他勇敢地成为英属缅甸第一位因发表政治演说而被监禁的人，为公民不服从树立了一个强有力的先例，并激励了无数人克服对殖民当局的恐惧。[^77]

学术研究人员赞扬吴·奥塔玛能够主要动员政治活跃的僧侣和农村人口的支持，从而创建一个超越教育和阶级界限的广泛运动。[^78] 以甘地的方式，他将一个基本的政治问题——缅甸的民族主义和独立——转化为一个宗教问题，吸引了甚至那些未受过英国教育的人，使得独立运动对所有缅甸人民都可及。[^79]

### 消极评价

批评者指出吴·奥塔玛方法中固有的矛盾，特别是他声称致力于非暴力与他关于独立“必要时可以通过暴力手段”获得的说法之间的紧张关系。[^80] 他们认为，这种哲学上的不一致性破坏了他信息的连贯性，并助长了民族主义运动中的暴力。

传统的佛教权威批评吴·奥塔玛和其他积极参与政治的僧侣违反了禁止僧侣参与世俗政治的戒律。[^81] 缅甸僧伽的大多数资深僧侣反对这种参与，认为这是对僧侣道路的腐蚀，并分散了对精神实践的注意力。[^82] “僧侣的政治煽动‘并没有得到广大民众的认可，因为这种公开参与反殖民政治或社会活动被视为违反了僧侣戒律。’”[^83]

一些历史学家认为，吴·奥塔玛的世界主义、现代化方法在动员缅甸农村人口方面的效果不如更传统的方法。[^84] 学者罗伯特·所罗门指出，吴·奥塔玛是一位“世界主义的革命家”，而萨亚·桑的努力“不那么现代但更成功”，因为1930年的缅甸农村人口对象征性诉求比对改革的计划性设计反应更强烈。[^85] 这表明吴·奥塔玛的国际教育和接触西方政治思想可能实际上限制了他对普通缅甸村民的影响力。

批评者还指出，在他生命的最后几年，吴·奥塔玛遭受了“贫困和被年轻一代缅甸民族主义者忽视”，这表明在他去世时，他的方法和领导力已经被取代。[^86] 昂山等更世俗的民族主义领导人的出现代表了对吴·奥塔玛以宗教为中心的独立方法的转变，尽管这些后来的领导人是在他所奠定的基础上进行的。[^87]

殖民当局及其辩护者将吴·奥塔玛仅仅视为“黄色僧袍中的政治煽动者”，否认他宗教信仰的真诚性，并将他描绘成一个机会主义者，利用他的僧侣身份谋取政治利益。[^88] 然而，这种定性普遍被后殖民历史学家所拒绝，认为这是试图否定真正的反殖民抵抗。

| 1901-1906 | 在加尔各答大学担任巴利语和佛教文学讲师；游历印度、法国和埃及 |
| 1907 | 1月前往日本；在东京佛教科学学院教授巴利语和梵语 |
| 1907-1910 | 广泛游历朝鲜、满洲、旅顺、中国、安南、柬埔寨、泰国、斯里兰卡 |
| 1910年代 | 返回缅甸；在瑞泽迪寺院学习；正式剃度出家，法号“奥塔玛” |
| 1920 | 开始政治活动；游历缅甸为青年佛教徒协会发表演讲 |
| 1920-1921 | 在缅甸各地创立“Wunthanu Athins”（爱国社团）；为《太阳报》撰稿 |
| 1921 | 因发表反对总督克拉多克的“克拉多克，滚出去！”演讲而被捕；首次因煽动叛乱罪被监禁 |
| 1921-1927 | 因煽动叛乱罪被反复监禁；在此期间入狱时间多于出狱时间 |
| 1924 | 由于他的影响，“Wunthanu Athins”几乎遍布缅甸每个村庄 |
| 1929 | 6月代表印度国民大会党参加孙中山先生的葬礼 |
| 1930-1932 | 萨亚·桑起义发生（承认吴·奥塔玛早期对萨亚·桑的影响） |
| 1935 | 担任全印度印度教大会的领导人（他唯一的正式职位） |
| 1936-1939 | 最后几年因疾病、糖尿病以及多年监禁导致的健康状况下降 |
| 1939 | 9月9日因糖尿病并发症在仰光医院去世，享年60岁 |

## 名言

> “克拉多克，滚出去！” - 吴·奥塔玛1921年反对总督雷金纳德·克拉多克爵士提出的克拉多克计划的著名演讲，该演讲导致他首次因煽动叛乱罪被捕入狱。[^89]

> “英国统治是佛教实践的障碍，因此必须获得独立。” - 吴·奥塔玛将宗教自由与政治独立联系起来的论点，构成了他融合佛教和民族主义思想的基础。[^90]

> “在当时，缅甸许多人甚至害怕最低级的英国官员，吴·奥塔玛激昂的言论具有激励人们、消除在帝国主义面前的恐惧的力量。” - 对吴·奥塔玛演讲对缅甸独立运动影响的历史评价。[^91]
