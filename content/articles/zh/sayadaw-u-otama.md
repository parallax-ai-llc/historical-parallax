## 摘要

赛亚道·吴·奥塔玛 (Sayadaw U Ottama) (1879-1939) 是一位上座部佛教僧侣，他成为缅甸在英国殖民统治下独立运动的先驱领袖。他被称为“缅甸的甘地”，是第一位踏入政坛的僧侣，也是英属缅甸第一位因发表政治演讲而被监禁的人。受甘地非暴力抵抗原则的启发，吴·奥塔玛走遍全国，建立民族主义组织，鼓吹抵制英货，并动员僧侣和普通民众反抗殖民压迫。在1921年至1927年间，他入狱的时间比在狱外的时间还多，他不知疲倦的积极活动为缅甸最终于1948年获得独立奠定了基础。[^1][^2][^3]

---

## 早年生活和背景

吴·奥塔玛出生时名为波·吞·昂 (Paw Tun Aung)，于1879年12月28日出生在缅甸西部若开邦（今实兑）附近鲁帕村。[^4] 他是吴·貌 (U Mra) 和杜·昂·觉·比 (Daw Aung Kwa Pyu) 的儿子，家境贫寒，生活艰难。[^5] 尽管出身卑微，奥塔玛和他的兄弟觉·吞·昂（后以阿里亚 (Ariya) 闻名）都曾在英国人开办的乡村学校接受了基础教育，年幼的奥塔玛显示出聪慧的学业天赋。[^6]

为了接受更深入的宗教教育，年少的波·吞·昂化装前往邻近马圭地区的著名帕科库寺院，并加入了僧团。[^7] 奥塔玛从小就展现出非凡的智力，在寺院里学业优异，赢得了许多高僧和认识到他潜力的经济赞助人的青睐。[^8] 这些赞助人帮助他申请加尔各答大学，这为他日后成为宗教领袖和政治革命家打开了大门。[^9]

十九岁时，他完成了入学考试，前往印度，并在加尔各答大学学习了数年，专注于佛教研究、巴利语和梵语。[^10] 许多年来，奥塔玛在加尔各答大学担任巴利语和佛教文学讲师，展现了他与生俱来的语言天赋，并迅速成为一名通晓多国语言的学者，学习并教授梵语、英语、日语以及许多其他语言。[^11] 后来他返回缅甸，正式受戒成为一名僧侣，法号为奥塔玛，在巴利语中意为“至高”或“崛起”。[^12] 他在帕科库活动了一段时间，之后返回若开邦，在瑞寨寺 (Shwezedi Monastery) 学习，并帮助他的兄弟觉·吞·昂加入僧团。[^13]

## 崛起

吴·奥塔玛从学者型僧侣转变为革命领袖，始于他20世纪初期的广泛国际旅行。在加尔各答学习三年后，他游历了印度，接触到了蓬勃发展的印度独立运动。[^14] 在印度，他与印度国民大会党和圣雄甘地关系密切，甘地的非暴力抵抗哲学对奥塔玛的反殖民斗争方式产生了深远影响。[^15] 他还访问了法国和埃及，拓展了他的世界观，并对反殖民运动有了更深的理解。[^16]

1907年1月，吴·奥塔玛前往日本，并在东京的佛教科学学院教授巴利语和梵语。[^17] 这使他成为首批在国外学习和教学的缅甸僧侣之一。[^18] 之后，他广泛游历了朝鲜、满洲、旅顺、中国、安南、柬埔寨、泰国、斯里兰卡和印度，以及美国和英国。[^19] 这些旅行让他接触了各种独立运动和政治思想，他后来将这些带回缅甸，帮助领导反抗英国统治的反殖民斗争。[^20]

返回英属缅甸后，吴·奥塔玛开始了他的政治活动，巡游全国，为青年佛教协会 (YMBA) 演讲，发表反殖民演说，吸引了大量追随者。[^21] 他成为YMBA的成员，为民族主义运动注入了新的活力，缅甸佛教僧侣（Pongyis）中很大一部分有影响力的群体都受到了他的影响。[^22] 作为一位激烈的演说家和鼓动者，吴·奥塔玛吸引了大量追随者，主要是佛教僧侣，他们组织了示威和集会，这引起了英国殖民当局的警惕，他们采取强硬手段，出动军警驱散集会。[^23]

吴·奥塔玛在他到访的每个地方都创立了“民生社”(Wunthanu Athins)（爱国社团或民族主义协会）。[^24] 到1924年，被称为“民生社”的组织几乎遍布缅甸的每个村庄，在反对英国殖民统治的政治动员中发挥了重要作用。[^25] 数千个此类协会在地方层面运作，负责弘扬缅甸传统、民族认同和佛教的自豪感，并鼓动反对1907年《村庄法》等压迫性措施。[^26] 吴·奥塔玛积极参与新兴政治运动，为民族主义报纸《太阳报》(Thuriya) 撰稿，在村庄里讲道，并建立“民生社”分支机构。[^27]

## 主要成就

吴·奥塔玛最显著的成就是他在缅甸独立斗争中开创性地将佛教僧侣身份与政治活动相结合。奥塔玛是首批进入政坛的僧侣之一，也是英属缅甸第一位因发表政治演讲而被监禁的人，随后有昂山和吴努等长长的民族主义者追随。[^28] 许多人认为他是该国独立运动之父，并被视为缅甸民族主义第一位真正的烈士和现代若开民族主义运动之父。[^29]

受甘地原则的启发，吴·奥塔玛倡导非暴力抵抗，提倡和平抗议、抵制英货以及复兴本土价值观和自力更生。[^30] 他以一种甘地式的方式，将一个基本上是政治问题——缅甸的民族主义和独立——转化为一个宗教问题，即使是那些未受英国教育的人也对其产生了共鸣。[^31] 他通过佛教伦理来表达对殖民统治的反对，创造了一个强有力地引起缅甸人民共鸣的信息。[^32] 这种将宗教身份融入政治活动的独特关注，建立了与民族主义斗争深厚的社区联系。[^33]

他最著名的反抗行为发生在1921年，当时他发表了臭名昭著的“克拉多克，滚出去！”演讲，反对当时英属缅甸总督雷金纳德·克拉多克爵士提出的克拉多克计划。[^34] 这次演讲导致他被捕并因煽动叛乱罪入狱，使他成为英属缅甸第一位因发表政治演讲而被监禁的人。[^35] 在当时缅甸许多人甚至害怕最低级别的英国官员时，吴·奥塔玛激烈的言辞有力地激励民众，并在帝国主义面前消除了恐惧，因此，全国各地的民族主义协会蓬勃发展，僧侣们开始发表反殖民演说。[^36]

吴·奥塔玛在推动缅甸民族主义运动转型方面发挥了关键作用，他提倡非暴力和抵制，让人联想到甘地的策略，他的影响导致到1924年缅甸农村各地建立了广泛的民族主义协会。[^37] 受奥塔玛的启发，大量佛教僧侣参与了政治运动。[^38] 他强调与英国改革进行不合作和抵制，而非起义，提倡和平的政治变革途径。[^39] 他通过演讲和在《太阳报》上的爱国文章激励了缅甸人，该报支持他，并在没有寺院接纳他作为政治僧侣时为他提供了媒体平台。[^40]

他非凡的贡献为他赢得了“亚洲之阳”的称号，他的爱国热情和勇气激励了缅甸年轻一代，他们最终在1948年赢得了独立。[^41] 1929年，他代表印度国民大会党参加了孙中山博士的葬礼，这显示了他的泛亚联系以及国际上对其领导地位的认可。[^42]

## 争议和批评

尽管作为民族主义英雄备受尊敬，吴·奥塔玛的政治活动并非没有争议，特别是关于佛教僧侣在政治中的作用。僧侣的政治煽动“并未被广大民众普遍接受，因为这种公开参与反殖民政治或社会活动被视为违反了僧侣戒律”。[^43] 缅甸僧伽的多数高级僧侣都反对僧侣参与政治，认为这类活动有悖于约束僧侣行为的戒律。[^44]

学者迈克尔·门德尔森 (Michael Mendelson) 写道，所有参与政治活动的僧侣都倾向于被殖民当局贴上“黄袍下的政治煽动者”的标签，这种定性试图削弱他们的宗教权威。[^45] 英国殖民政府视吴·奥塔玛为危险的激进分子，并以煽动叛乱罪多次将其监禁。[^46] 据学者称，在1921年至1927年间，吴·奥塔玛入狱的时间比在狱外的时间还要多，他忍受了恶劣的条件，这些条件最终损害了他的健康。[^47]

吴·奥塔玛的哲学方法中也存在矛盾。尽管吴·奥塔玛忠于他的甘地主义理想，并且本人并不提倡在反殖民斗争中使用暴力，但他认为英国统治是佛教实践的障碍，因此必须通过包括暴力在内的手段来实现独立。[^48] 为了支持在特定情况下的使用武力，他引用了一些《本生经》(Jatakas)（佛教佛陀前世的故事），在他声称的非暴力与反殖民抵抗的现实之间造成了紧张关系。[^49] 尽管他个人倡导非暴力，但在20世纪20年代，发生了几起涉及激进僧侣反对殖民统治的暴力事件，其中一些人声称受到吴·奥塔玛教导的启发。[^50]

在他晚年，吴·奥塔玛遭受疾病、贫困和被年轻一代缅甸民族主义者忽视的境遇，这些年轻一代已经超越了他的方法。[^51] 一些历史学家指出，尽管吴·奥塔玛在20世纪20年代初确实影响了萨亚·桑 (Saya San)，但这两位领导人之间的差异比相似之处更大。[^52] 吴·奥塔玛被描述为一位“国际主义革命家”，而萨亚·桑的努力“不太现代化但更成功”，通过传统的宗教呼吁而非现代民族主义纲领动员了农村农民。[^53]

## 个人生活

在其职业生涯中，吴·奥塔玛始终保持着佛教僧侣简朴的生活，即使在他成为杰出的政治人物后，也坚守着独身和简朴的戒律。他出生于一个贫困的若开家庭，从未忘记自己卑微的出身，并始终致力于为缅甸普通民众服务。[^54]

他非凡的语言能力使他迅速成为一名通晓多国语言的学者，学习并教授梵语、英语、日语以及许多其他语言。[^55] 这种语言能力使他既能成为一名优秀的教师，又能成为一名能够与亚洲各地不同听众交流的政治组织者。他的兄弟觉·吞·昂（法号阿里亚）在瑞寨寺与他一起加入了僧团，尽管历史记录对他们之间的关系或阿里亚的活动鲜有细节。[^56]

吴·奥塔玛以其激昂的演说和富有魅力的形象而闻名。作为一位激烈的演说家和鼓动者，他吸引了大量追随者，主要是佛教僧侣，他们组织了示威和集会来支持他。[^57] 他的演讲对听众产生了深刻的情感影响，在英国殖民势力似乎不可战胜的时候，消除了他们的恐惧，激励他们采取行动。[^58]

在1921年至1927年期间，他反复入狱，忍受了巨大的苦难。英国殖民当局忽视了他的非暴力方式，并对他进行了残酷的对待，导致他的健康状况恶化。[^59] 在他生命的最后几年，他患有糖尿病以及其他与多年监禁和持续旅行及活动带来的身体损耗相关的疾病。[^60] 《太阳报》支持他，并在没有寺院接纳他作为政治僧侣时为他提供了平台，这表明他的活动即使在佛教圈内也存在争议。[^61]

## 遗产

吴·奥塔玛于1939年9月9日因糖尿病在仰光医院去世，享年60岁，这是多年监禁和健康状况下降的结果。[^62] 他的去世标志着缅甸独立运动一个时代的结束，但他的影响将在未来几十年继续塑造国家。

吴·奥塔玛的活动促成了一个激进的反殖民运动的出现，该运动融合了民族主义、佛教，并在后期加入了社会主义。[^63] 这种政治和宗教信仰的结合，在殖民时期的缅甸的大多数民族主义政治家都不同程度地遵循——在1948年国家独立后，这一点几乎没有改变。[^64] 他的工作为昂山和吴努等后来的民族主义领导人铺平了道路，为缅甸最终在1948年实现独立做出了贡献。[^65]

他于1939年9月9日的逝世，每年都被纪念为“吴·奥塔玛日”，以纪念他的逝世，并提醒当今世代缅甸人民为反抗殖民压迫所付出的长期斗争。[^66] 他的忌日广泛在若开邦各地（包括皎漂、丹兑等乡镇）以及泰国、马来西亚、韩国和其他有缅甸侨民社区的国家举行纪念活动。[^67] 他在现在的若开邦实兑的旧寺院瑞寨寺，至今仍然是政治和反政府活动的重要中心。[^68]

吴·奥塔玛被认为是现代缅甸的民族英雄之一，他对独立斗争的贡献在官方历史和公众记忆中都得到了认可。[^69] 他既是一位宗教人物，也是一位政治革命家，他证明了僧侣们可以在保持其精神承诺的同时，积极参与反对不公正的斗争。[^70] 他的榜样激励了缅甸一代又一代的政治活跃僧侣，尽管僧侣是否应该参与政治的问题至今仍然存在争议。[^71]

## 历史评价

### 积极评价

学者和历史学家们普遍认为吴·奥塔玛是一位变革性人物，他成功地将佛教宗教身份与政治民族主义相结合，开创了塑造缅甸独立运动数十年的方法。[^72] 他通过提倡非暴力和抵制（让人联想到甘地的策略）在推动缅甸民族主义运动转型方面发挥了关键作用，他的影响导致到1924年缅甸农村各地建立了广泛的民族主义协会。[^73]

学术界的共识是将吴·奥塔玛描绘成一位奠基性人物，他独特的专注于将宗教身份融入政治活动的视角，为民族主义斗争建立了深厚的社区联系。[^74] 在政治意识浓厚的环境中成长并在印度学习，塑造了吴·奥塔玛的非暴力抵抗方法，让人联想到甘地的原则，他深受与印度民族主义运动和佛教教义的接触的影响。[^75]

人权观察和其他国际组织已将吴·奥塔玛视为缅甸佛教活动主义传统的先驱人物，指出他是许多反抗英国殖民统治的僧侣中的第一位。[^76] 他作为英属缅甸第一位因发表政治演讲而被监禁的人的勇气，为公民不服从树立了一个有力的先例，并激励了无数人克服了对殖民当局的恐惧。[^77]

学术研究者赞扬吴·奥塔玛能够激发政治活跃僧侣和农村民众的支持，创造了一个超越教育和阶级界限的广泛运动。[^78] 他以甘地式的方式，将一个基本上是政治问题——缅甸的民族主义和独立——转化为一个宗教问题，即使是那些未受英国教育的人也对其产生了共鸣，从而使所有缅甸人都能接触到独立运动。[^79]

### 负面评价

批评者指出吴·奥塔玛方法中固有的矛盾，特别是他声称致力于非暴力与他认为独立“必要时可以通过暴力手段”获得的说法之间的紧张关系。[^80] 他们认为，这种哲学上的不一致削弱了他信息的连贯性，并导致了民族主义运动中的暴力。

传统的佛教权威批评吴·奥塔玛和其他参与政治活动的僧侣违反了禁止僧侣参与世俗政治的戒律。[^81] 缅甸僧伽的多数高级僧侣都反对这种参与，认为这腐蚀了僧侣的道路，并分散了对精神实践的关注。[^82] 僧侣的政治煽动“并未被广大民众普遍接受，因为这种公开参与反殖民政治或社会活动被视为违反了僧侣戒律”。[^83]

一些历史学家认为，吴·奥塔玛的国际化、现代化方法在动员缅甸农村地区民众方面的效果不如更传统的方法。[^84] 学者罗伯特·所罗门 (Robert Solomon) 指出，吴·奥塔玛是一位“国际主义革命家”，而萨亚·桑的努力“不太现代化但更成功”，因为1930年的缅甸农村民众对象征性呼吁的反应比对改革的纲领性设计更强烈。[^85] 这表明吴·奥塔玛的国际教育和对西方政治思想的接触可能实际上限制了他与普通缅甸村民的有效性。

批评者还指出，在他晚年，吴·奥塔玛遭受了“贫困和被年轻一代缅甸民族主义者忽视”，这表明在他去世时，他的方法和领导力已被取代。[^86] 昂山等更世俗化的民族主义领导人的出现，标志着远离吴·奥塔玛以宗教为中心的独立方式，尽管这些后来的领导人是在他奠定的基础上进行发展的。[^87]

殖民当局及其支持者将吴·奥塔玛贬低为仅仅是“黄袍下的政治煽动者”，否定了他宗教信仰的真诚性，并将他描绘成一个机会主义者，为了政治目的而利用其僧侣身份。[^88] 然而，后殖民历史学家普遍拒绝这种定性，认为这是一种试图否定真实的殖民反抗的企图。

| 1898 | 进入帕科库寺院，加入僧团（约略日期） |
| 1898-1901 | 在加尔各答大学学习三年 |
| 1901-1906 | 在加尔各答大学担任巴利语和佛教文学讲师；游历印度、法国和埃及 |

| 1910年代 | 返回缅甸；在瑞寨寺学习；正式受戒为僧，法号“奥塔玛” |
| 1920 | 开始政治活动；巡游缅甸为青年佛教协会演讲 |
| 1920-1921 | 在缅甸各地创立“民生社”（爱国社团）；为《太阳报》撰稿 |
| 1921 | 因反对总督克拉多克提出的“克拉多克计划”的演讲而被捕；首次因煽动叛乱罪入狱 |

| 1924 | 由于他的影响，“民生社”几乎遍布缅甸的每个村庄 |
| 1929 | 6月代表印度国民大会党参加孙中山博士的葬礼 |
| 1930-1932 | 发生萨亚·桑起义（承认吴·奥塔玛早期对萨亚·桑的影响） |
| 1935 | 担任全印度印度教大会 (All India Hindu Mahasabhas) 的领导人（他担任的唯一正式职务） |
| 1936-1939 | 晚年因多年监禁导致的疾病、糖尿病和健康状况下降 |
| 1939 | 9月9日因糖尿病相关疾病在仰光医院去世，享年60岁 |

## 名言

> “克拉多克，滚出去！” - 吴·奥塔玛1921年反对总督雷金纳德·克拉多克爵士提出的克拉多克计划的著名演讲，该演讲导致他首次因煽动叛乱罪被捕入狱。[^89]

> “英国统治是佛教实践的障碍，因此必须获得独立。” - 吴·奥塔玛将宗教自由与政治独立联系起来的论点，构成了他融合佛教和民族主义意识形态的基础。[^90]

> “在当时缅甸许多人甚至害怕最低级别的英国官员时，吴·奥塔玛激烈的言辞有力地激励民众，并在帝国主义面前消除了恐惧。” - 对吴·奥塔玛对缅甸独立运动的演讲影响的历史评价。[^91]
