## 摘要

李範奭（이범석；1900年10月20日－1972年5月11日），亦以艺名“铁骥”（鐵驥）闻名，是韩国一位重要的独立活动家、军事领导人和政治家，曾于1948年至1950年担任韩国首任总理[^1]。作为世宗大王的后裔，李範奭毕生致力于武装反抗日本殖民统治，参与了1920年青山里战役等著名战役，并在第二次世界大战期间升至韩国光复军总司令[^2]。他的遗产依然极具争议，他因对韩国独立运动的贡献而受到赞颂，但也因其独裁倾向和涉嫌的法西斯同情而受到批评[^3]。

---

## 早年生活与背景

李範奭于1900年10月20日出生在朝鲜王朝末期的京城（今首尔）。[^4] 他的父亲曾是朝鲜政府的一名官员，李範奭属于全州李氏，是世宗大王之子广平大君的直系后裔，属于韩国的贵族两班阶层。[^5] 这种高贵的血统后来影响了他的民族主义思想和他历史使命感。

1915年，年仅十五岁的李範奭做出了一个改变人生的决定，他流亡中国，投身于反日独立活动，离开了家人和祖国。[^6] 同年，他与当时忠清南道天安郡守金承铉的女儿订婚，但这段婚姻并未长久，因为李範奭将独立斗争置于家庭生活之上。[^7] 这种早期对武装抵抗的承诺决定了他的一生轨迹，将他从一个贵族青年转变为一名革命士兵。

李範奭的革命战争正规教育始于1919年，他进入了满洲的新兴军事学校。[^8] 这所学校成立于1911年，其明确使命是培训韩国人进行武装反抗日本占领的斗争。[^9] 该学校实施严格的军事训练，教授战术、骑术和游击战，以使学员为实战做好准备。[^10] 到1919年10月，李範奭已从学生晋升为学校教官，指导下一代独立斗士。[^11] 新兴军事学校在1920年关闭前，培养了约2100名校友，其中许多人成为韩国独立运动的重要人物。[^12]

## 崛起

李範奭成为一位杰出的军事领导人，是在韩国抵抗运动历史上最受颂扬的战役之一中实现的。1920年3月，他接到了北方军政府（북로군정서）传奇指挥官金佐镇的召唤，进行了数千英里的艰苦徒步行军，加入了金佐镇在满洲汪清县的部队，并被任命为训练官（연성대장）。[^13]

他决定性的时刻出现在1920年10月的青山里战役（又称青山里大捷）中，这是一场在满洲东部持续六天的战斗，成为韩国对日军取得的最重要的胜利之一。[^14] 李範奭担任第二大队后卫指挥官，带领他的600名士兵在白云坪（백운평）峡谷的伏击阵地上，隐藏在茂密森林地形的岩石缝隙中。[^15] 1920年10月20日——碰巧也是他20岁的生日——激烈的战斗爆发了，李範奭的部队成功地在一次突袭中歼灭了日军先头部队。[^16] 在他的回忆录《乌冬不》（우둥불）中，李範奭声称日军伤亡人数为3300人，尽管历史学家普遍认为其回忆录中的数字存在争议。[^17] 尽管如此，青山里大捷的消息振奋了海外朝鲜人社群，并确立了李範奭作为一位强大军事指挥官的声誉。当时作为韩国独立外交倡导者的李承晚，在听到胜利的消息后写信给李範奭，表示：“现在我有什么可以向全世界吹嘘的了。凭借青山里的大捷，我获得了重要的宣传材料，使我能够进行不撒谎的外交。”[^18]

战役结束后，李範奭和他的妻子金玛利亚于1925年被苏联红军在托木斯克俘获，被拘留了八个月，因为苏联试图解除朝鲜独立武装力量的武装。[^19] 李範奭反对苏联的解除武装政策，并于1926年4月，他与金玛利亚成功逃脱拘留，逃往满洲的会宁地区。[^20]

到1941年，李範奭已晋升为上将，并担任驻重庆的韩国临时政府的军事部门——韩国光复军（KLA）的总参谋长。[^21] 在此职位上，他直接与美国情报机构进行了谈判，于1944年9月会见了美国军事情报局的约瑟夫·迪基上校和战略情报局（OSS）的克莱德·贝利·萨金特上尉。[^22] 这些会议促成了“猎鹰计划”（독수리작전），这是OSS与KLA之间的一项联合行动，于1945年3月13日获得批准，旨在训练韩国特种兵并将其渗透到日本占领的五个战略城市：清津、新义州、釜山、平壤和汉城。[^23] 尽管日本于1945年8月15日投降，阻止了该行动的执行，但“猎鹰计划”显示了李範奭的战略重要性和国际外交能力。[^24]

## 主要成就

在流亡数十年后于1946年返回韩国，李範奭立即在新兴的韩国国家中确立了他作为关键政治和军事人物的地位。[^25] 他与安浩相一起，于1946年10月9日创立了韩国青年团（대한청년단；KNYA），这是一个极右翼民族主义组织，成为早期韩国最有影响力的政治力量之一。[^26] KNYA由美国军事政府赞助，其意识形态融合了法西斯主义和抵抗民族主义，拒绝西方资本主义和苏联共产主义，转而支持以朝鲜民族认同为中心的第三世界主义民族主义。[^27]

李範奭与李承晚结盟，对抗金九关于南北谈判的提议，支持李承晚在韩国建立独立政府的设想。[^28] 在1948年5月10日制宪议会选举和1948年7月17日通过韩国第一部宪法后，李承晚于7月20日当选总统。[^29] 尽管李範奭并非李承晚的首选——李承晚最初提名的李允荣于7月27日被国民议会否决——但议会于1948年7月29日批准了李範奭的提名，他于1948年7月31日正式就任，成为韩国首任总理。[^30]

从1948年8月15日至1949年3月，李範奭兼任首任国防部长，他领导了韩国军事史上最关键的转型之一：将韩国陆军（于1946年1月15日在美国军事政府下成立，约有2.5万名人员）转变为正式的韩国陆军（ROKA），于1948年9月5日正式成立。[^31] 在他的领导下，韩国陆军扩展到约5万名士兵，并组织成五个师。[^32] 李範奭战略性地将70多名韩国光复军老兵整合到关键指挥职位，确保新军事领导层由久经沙场、思想坚定、反共的军官组成。[^33]

李範奭担任总理的时间从1948年7月31日持续到1950年4月20日，这段时期在国内混乱的解放后条件下，他致力于巩固国家权力、镇压左翼运动和建立政府机构。[^34] 他的领导风格强调纪律、青年动员和军事民族主义，反映了他数十年的军事经验和他对有组织力量作为国家力量基础的信念。[^35]

## 争议与批评

李範奭的政治生涯引发了激烈的争议，批评者指责他具有违背民主原则的独裁和准法西斯倾向。他创立并领导的韩国青年团获得了“李範奭希特勒青年团”（이범석 유겐트）的贬称，明显将其与希特勒青年团相提并论。[^36] 政治反对派称他为韩国的“准军国主义者”和“希特勒追随者”，这些指控基于李範奭在1945年得知希特勒死亡和纳粹德国覆灭时表示遗憾的报道。[^37] 尽管这些言论的真实性和背景仍有争议，但它们严重损害了他在民主倡导者中的声誉。

他的政治意识形态融合了强调军事力量和青年动员的社会达尔文主义历史观，以及他在20世纪30年代末期在中国国民党（ Kuomintang）期间吸收的极权主义和反帝国主义思想，当时国民党政府采纳了受法西斯主义影响的政策。[^38] 批评者认为李範奭未能正确理解或拥抱自由民主，而是将国家和政权置于个人权利和自由之上。[^39] 韩国青年团本身也被批评为一个专制、雇佣兵式的组织，为李範奭的政治野心服务，而非真正促进民主发展。[^40]

在李範奭担任总理期间，由于在治理和军事控制方面的愿景不同，他与李承晚总统的关系急剧恶化。[^41] 李承晚是一位专注于集中行政权力的文职领导人，他越来越警惕李範奭广泛的军事背景、个人忠诚网络以及与独立运动派别的联系。[^42] 这种紧张关系促成了李範奭于1950年4月20日辞职，就在朝鲜战争爆发前几个月。

李範奭此后多次竞选均告失败。他分别于1952年和1956年竞选副总统，但两次均落败。[^43] 在1952年的竞选中，总理张泽相——李範奭曾与其争夺李承晚的青睐——秘密指示自由党成员支持咸台永而非李範奭，这导致了李範奭的失败。[^44] 随后，李範奭起诉张泽相，声称警方干预了选举。[^45] 这些选举失利使李範奭在20世纪50年代的关键时期被边缘化，尽管他Throughout the 1960s, he remained active in opposition politics[^46].

## 个人生活

李範奭于1925年8月与金玛利亚（김마리아；1903-1970）结婚，两人结成了经受住革命艰难和政治动荡的伴侣关系。[^47] 金玛利亚本人也是一位女性独立活动家，曾积极参与反日抵抗活动。[^48] 他们的关系是在独立斗争的熔炉中建立起来的；结婚后不久，两人于1925年被苏联军队在托木斯克一同俘获并拘留了八个月，直到1926年4月才成功逃脱。[^49]

与他那个时代的许多政治人物不同，李範奭保持着相对私人的家庭生活，关于他的子女或后代的公开信息非常有限。他早期于1915年与金承铉的女儿订婚，但随着李範奭前往中国加入独立运动而解除，这表明他的革命承诺完全超越了传统的家庭期望。[^50]

李範奭的性格融合了军事纪律和民族主义热情。他数十年的游击战、军事教育和流亡经历塑造了他的世界观，强调力量、牺牲和民族主义。他的艺名“铁骥”（鐵驥）反映了他这种军事身份。[^51] 同时代人形容他意志坚定、思想坚定且生活简朴，这些品质使他成为一名成功的抵抗战士，但也为他在民主治理方面带来了政治上的不利。

金玛利亚于1970年2月1日在首尔新堂洞的家中因慢性糖尿病去世。[^52] 李範奭在她去世两年多后，于1972年5月11日逝世。[^53]

## 遗产

李範奭于1972年5月11日因心肌梗死在首尔明洞圣母医院去世，前一天（5月10日）他刚刚获得台湾中国文化学院的荣誉博士学位。[^54] 他的国葬于1972年5月17日在南山广场举行，并被安葬在首尔国立公墓，这是对他作为独立英雄和建国政治人物地位的认可。[^55]

李範奭对韩国历史的长远影响体现在多个方面。作为一名军事领导人，他帮助奠定了大韩民国陆军的制度基础，整合了独立运动的老兵，并将反共意识形态作为核心军事价值观。[^56] 他在青山里战役中的作用确保了他作为韩国民族记忆中反抗日本帝国主义的英雄地位。[^57] 韩国海军以潜艇“李範奭号”（ROKS Lee Beom-seok，SS-081）来纪念他，这反映了官方对其军事贡献的认可。[^58]

然而，李範奭的政治遗产仍然充满争议。他领导方式的独裁方面，他与极右翼民族主义运动的联系，以及涉嫌的法西斯同情，仍在持续引发历史争论。[^59] 近年来，韩国社会一直在努力重新评估早期独立活动家和建国政治人物，权衡他们反殖民的贡献以及他们后来的独裁行为。[^60] 这种争议在2023年显现，当时关于移除包括一些有争议人物在内的独立活动家雕像的讨论在韩国陆军士官学校出现，反映了如何纪念复杂历史人物的持续紧张关系。[^61]

在他职业生涯的后期，李範奭担任了国民统一审议会的顾问，并指导了朴正熙，朴正熙在1961年的军事政变中夺取了权力，并统治韩国直至1979年。[^62] 这种指导关系将李範奭这一代的独立战士与20世纪80年代主导韩国政治的军事独裁联系起来，这是一段复杂而深刻影响韩国政治文化的遗产。[^63]

## 历史评估

### 积极观点

学者和李範奭的辩护者强调他在韩国反抗日本殖民统治的武装独立运动中的关键作用，特别是他在1920年青山里战役中的领导作用，该战役至今仍是韩国最受颂扬的军事胜利之一。[^64] 他的贡献通过“建国勋章”（总统表彰）获得了追授认可，这是韩国对他在1915年流亡中国直至解放时期反殖民活动官方的认可。[^65] 支持者将他描述为一位“被遗忘的独立英雄”，他强调军事民族主义和武装抵抗，为纯粹的外交手段争取韩国独立提供了关键的平衡。[^66]

辩护者指出，李範奭数十年的军事服务——从他在新兴军事学校的青少年时期，到他担任韩国光复军总司令——以巨大的个人代价展示了他对韩国主权的坚定承诺。[^67] 他成功与美国战略情报局（OSS）达成“猎鹰计划”，展示了他的战略复杂性和国际外交能力，使韩国成为盟军战争努力的积极参与者，而非被动的受害者。[^68] 他在建立大韩民国陆军中的作用，为独立斗争与新韩国国家之间提供了制度连续性，确保了由解放战士而非殖民合作者控制军队。[^69]

一些历史学家将李範奭的独裁倾向置于解放后的混乱环境中，认为他强调纪律和组织是为了应对共产主义叛乱和政治分裂的真正威胁。[^70] 从这个角度看，他的失败反映了在一个分裂、饱受战争蹂躏的国家中，国家建设所面临的更广泛挑战，而非个人道德上的失败。[^71]

### 消极观点

批评者认为，李範奭涉嫌的法西斯同情和独裁手段从根本上损害了他的独立活动，并质疑他的民族主义意识形态是否真正服务于民主价值观，还是仅仅用韩国的独裁取代了日本的殖民统治。[^72] 他对纳粹德国战败表示遗憾的报道，将他的韩国青年团与希特勒青年团进行明确比较，以及他采纳的极权主义组织原则，都表明他与法西斯主义存在意识形态上的亲近性，这与民主建国相悖。[^73]

批评李範奭的学者强调，他的政治意识形态根植于社会达尔文主义，优先考虑军事力量和民族主义，未能理解自由民主的个人权利、军事权力民事控制和多元治理原则。[^74] 他在20世纪30年代末期与受法西斯主义影响的中国国民党政策的经历，塑造了他优先考虑国家权力而非个人自由的极权主义和反资本主义观点。[^75] 韩国青年团作为一个专制、军事化的组织，被指控进行政治暴力和恐吓，引发了对李範奭对民主规范承诺的质疑。[^76]

批评者还指出，李範奭的政治失败——他在与李承晚的冲突中辞去总理职务，在1952年和1956年两次竞选副总统均告失败，以及在20世纪50年代和60年代的关键时期被边缘化——表明韩国选民和政治精英认识到了他独裁方式的危险。[^77] 他指导朴正熙，后者在韩国实行了18年的军事独裁统治，这在批评者看来，代表了将独立时代军国主义与随后独裁统治联系起来的令人不安的遗产。[^78]

一些历史学家认为，考虑到他的意识形态亲近性，即使是李範奭的独立运动资格也值得审视，并质疑是否由法西斯民族主义意识形态驱动的反日殖民动机，与基于民主和人道主义价值观的抵抗具有同等的道德分量。[^79] 这种观点认为，应该批判性地审视李範奭的遗产，而不仅仅是无条件地颂扬，同时承认他反殖民的贡献和他的独裁倾向。[^80]

| 1944 | 9月，在重庆与美国战略情报局（OSS）谈判“猎鹰计划” |
| 1945 | 3月13日，“猎鹰计划”获批；8月15日，日本投降，行动未能执行 |
| 1946 | 返回韩国；10月9日，与安浩相共同创立韩国青年团 |
| 1948 | 7月31日，被任命为韩国首任总理；8月15日，兼任首任国防部长 |
| 1948 | 9月5日，监督大韩民国陆军成立 |
| 1949 | 3月，卸任国防部长 |
| 1950 | 4月20日，辞去总理职务；6月25日，朝鲜战争爆发 |
| 1950-1951 | 12月至9月，担任韩国驻中华民国（台湾）大使 |
| 1952 | 竞选副总统失败，败给咸台永 |
| 1956 | 再次竞选副总统失败 |
| 1960年代 | 作为反对派领导人活跃；担任国民统一审议会顾问；指导朴正熙 |
| 1970 | 妻子金玛利亚去世（2月1日） |
| 1972 | 5月10日，获得台湾中国文化学院荣誉博士学位；5月11日，因心肌梗死去世；5月17日，举行国葬 |

## 著名引言

> “现在我有什么可以向全世界吹嘘的了。凭借青山里的大捷，我获得了重要的宣传材料，使我能够进行不撒谎的外交。” - 李承晚致李範奭的信，1920年，青山里战役后[^80]

注：直接来自李範奭本人的引言在可用的英文资料中有限。他的回忆录《乌冬不》（우둥불）包含了他对独立斗争的第一手记述，尽管历史学家指出其在具体细节（如伤亡数字）上的可靠性存在疑问。[^81]
