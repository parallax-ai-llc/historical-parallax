## 简介

已以 Faker 的名义涵盖。

李相赫以其游戏ID **Faker** 在全球广为人知，被广泛认为是《英雄联盟》历史上最具影响力的职业选手之一。他与 T1（前身为 SK Telecom T1）组织联系最为紧密，尤其以精英级的 [中单](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%8D%95) 表现、顶尖水平的持久性以及跨越多个竞技时代的职业生涯而闻名。

本文以他的法定姓名记录了简洁的个人生平，并指向与更广为人知的“Faker”报道相同的 [主要公开来源](https://en.wikipedia.org/wiki/Lee_Sang_Hyeok)。

## 职业生涯时间线

| 年份 | 事件 |
|------|-------|
| 2013 | 在韩国顶级联赛首次亮相，并随 SK Telecom T1 崛起至国际声望。 |
| 2013 | 随 SKT 赢得《英雄联盟》世界锦标赛（首个世界冠军）。 |
| 2015 | 随 SKT 再次赢得世界锦标赛（第二个世界冠军）。 |
| 2016 | 随 SKT 再次赢得世界锦标赛（第三个世界冠军）。 |
| 2017 | 在 SKT/T1 持续国内外竞争的时期，继续担任核心人物。 |
| 2019 | 随着组织从 SK Telecom T1 更名，以 T1 品牌参赛。 |
| 2020 | 与 T1 保持长期合作，继续成为电竞界最知名的选手之一。 |
| 2022 | 继续在最高水平的比赛中竞争；因其稳定性和团队领导力而广受称赞。 |
| 2023 | 随 T1 达到了职业生涯的又一个重要巅峰，巩固了他持久和关键表现的声誉。 |
| 2024 | 继续活跃于 T1，延长了其在电竞领域异常长的精英级职业生涯。 |

## 打法和声誉

- **位置：** 主要为 [中单](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%8D%95)；历来以高超的操作技巧和广泛的英雄池闻名。
- **竞技身份：** 常被描述为能够适应不同版本（meta）的变化，并能根据队伍需求扮演 [Carry](https://en.wikipedia.org/wiki/Carry_(video_games)) 型或辅助型中单。
- **领导力：** 常被引用为在阵容变动时稳定军心的存在，以及其组织的长期代表人物。

## 传奇地位（高层级）

- 成为《英雄联盟》电竞界标志性人物，并极大地推动了该游戏的主流知名度。
- 在该游戏中“史上最佳”的争论中，常被作为 [标杆](https://en.wikipedia.org/wiki/Benchmark_(computing)) 被提及。
- 与典型的电竞职业生涯长度相比，他在顶尖水平上保持了异常长的相关性，这一点尤为突出。
