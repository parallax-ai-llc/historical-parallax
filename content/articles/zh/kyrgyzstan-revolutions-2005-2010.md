## 摘要

吉尔吉斯斯坦在后苏联历史中占有独特的地位：它是中亚唯一一个通过民众起义推翻了两任总统的国家。2005年，“郁金香革命”推翻了自独立以来一直执政的阿斯卡尔·阿卡耶夫。2010年，在他继任的五年后，接替他的库尔曼别克·巴基耶夫本人也在一次更为血腥的起义中被推翻。两次，愤怒的民众都冲进了政府大楼。两次，总统都逃离了国家。两次，改革的承诺都未能实现。[^1]

吉尔吉斯革命展示了民众抗议的力量和局限性。人民表明他们不会容忍无限期的威权统治、欺诈性选举和家族腐败。但每一次革命之后都伴随着不稳定、族裔暴力，并最终出现了另一位令民主希望破灭的领导人。吉尔吉斯斯坦一再发生的革命揭示了一个在寻找真正民主但却在建立可持续制度方面步履维艰的国家。[^2]

## 第一部分：郁金香革命（2005年）

### 背景：阿卡耶夫的衰落

**阿斯卡尔·阿卡耶夫**曾被视为中亚最自由派的领导人：

- 自1990年（独立前）起担任总统
- 物理学教授；最初具有改革形象
- 比邻国允许更多的媒体自由
- 被称为中亚的“民主岛屿”

**但到了2005年：**

- 执政15年
- 家族控制经济
- 镇压反对派
- 腐败猖獗
- 民主的假象被揭穿[^3]

### 导火索：欺诈性选举

**2005年2月-3月：** 议会选举

**问题：**

- 国际观察员发现存在重大欺诈
- 阿卡耶夫支持的候选人宣布获胜
- 反对派候选人被取消资格或受到骚扰
- 抗议活动始于南部地区

**火花：** 如果阿卡耶夫能够操纵议会，他就能无限期地延长他的统治——甚至可能指定一个继承人（他的儿子或女儿被提及）。[^4]

### 革命

**2005年3月：抗议蔓延**

**吉尔吉斯斯坦南部：**

- 贾拉拉巴德：反对派占领了政府大楼（3月20日）
- 奥什：类似的占领
- 南部地区实际上处于反对派控制之下

**3月23日：**

- 反对派控制了全国三分之二的地区
- 阿卡耶夫仍控制比什凯克

**3月24日：垮台**

**下午：**

- 15000多名抗议者聚集在比什凯克阿拉套广场
- 游行至政府大楼
- 警察抵抗瓦解
- **抗议者冲进白宫（总统行政大楼）**
- 发生抢劫

**阿卡耶夫逃亡：**

- 乘直升机前往哈萨克斯坦
- 后前往俄罗斯
- 再未返回

**伤亡：** 3人死亡（在抢劫后续事件中），数十人受伤[^5]

### 后果

**临时政府：**

- **库尔曼别克·巴基耶夫：** 被任命为代理总统
- **菲利克斯·库洛夫：** 被释放出狱；成为安全主管
- 北部和南部之间存在不稳定的联盟

**2005年4月4日：** 阿卡耶夫在莫斯科签署辞职书

**2005年7月10日：** 举行选举

- 巴基耶夫以88.9%的得票率获胜
- 最初承诺进行民主改革[^6]

## 第二部分：2010年革命

### 巴基耶夫的背叛

**期望与现实：**

| 承诺            | 现实                         |
| -------------- | ---------------------------- |
| 民主改革       | 威权主义加剧                 |
| 结束家族统治   | 巴基耶夫家族控制经济         |
| 清廉政府       | 腐败泛滥                     |
| 媒体自由       | 媒体限制收紧                 |

**到2010年的关键问题：**

- 儿子马克西姆·巴基耶夫：被指控控制国家财政
- 能源和公用事业价格急剧上涨
- 政治反对派被逮捕
- 公民社会受到限制

**2010年：** 与2005年前的不满情绪几乎是惊人的重演[^7]

### 导火索：经济困境

**2010年初：**

- 电价上涨170%
- 供暖成本翻倍
- 燃油价格上涨
- 许多人无力负担基本公用事业

**对于普通吉尔吉斯人来说，这是生存问题。** 明显的原因是使巴基耶夫的亲信受益的私有化交易。[^8]

### 革命

**2010年4月6日：塔拉斯**

- 抗议活动在北部城镇塔拉斯开始
- 政府大楼被占领
- 州长被短暂劫持为人质
- 暴力蔓延

**2010年4月7日：比什凯克**

**上午：**

- 首都发生大规模抗议
- 数万人聚集
- 要求巴基耶夫辞职

**下午：**

- 部署了警察和安全部队
- 使用了催泪瓦斯、橡胶子弹
- **随后是实弹射击**
- 抗议者冲进了国内安全总部
- 占领了国家电视台

**死亡人数：**

- 官方：87人死亡
- 一些估计：90+
- **数百人受伤**

**比2005年血腥得多。**

**巴基耶夫逃亡：** 先逃往吉尔吉斯斯坦南部，然后是哈萨克斯坦，最后是白俄罗斯[^9]

### 为什么2010年更血腥？

**关键区别：**

| 2005年                      | 2010年                          |
| ---------------------------- | ------------------------------- |
| 安全部队未开枪               | 安全部队向抗议者开枪             |
| 阿卡耶夫迅速离开             | 巴基耶夫进行抵抗                 |
| 3人死亡                      | 87+人死亡                       |
| 相对有序的过渡               | 随后是混乱                     |

**巴基耶夫从阿卡耶夫的垮台中吸取了教训——但应用了错误的教训。** 他认为武力会奏效。但事实并非如此。[^10]

### 后果与族裔暴力

**临时政府：**

- **罗莎·奥通巴耶娃：** 成为临时总统
- 中亚第一位女性国家元首
- 承诺进行真正的民主改革

**2010年6月：奥什暴力事件**

**灾难：**

- 吉尔吉斯斯坦南部吉尔吉斯族和乌兹别克族之间发生族裔冲突
- 470多人死亡（多为乌兹别克族）
- 40万人流离失所
- 后苏联中亚最严重的族裔暴力事件

**原因争议：**

- 巴基耶夫的支持者被指控煽动
- 既有的族裔紧张关系
- 革命后的国家崩溃

**2010年革命的后果比其前身更为黑暗。**[^11]

## 历史评估

### 为什么发生两次革命？

**吉尔吉斯斯坦的模式：**

1. 反对派领导人通过抗议上台
2. 承诺改革
3. 巩固权力，使家族致富
4. 镇压反对派
5. 欺诈性选举或经济危机引发起义
6. 循环重复

**根本问题：**

- 制度薄弱
- 地区（南北）分歧
- 基于恩庇的政治
- 没有强大的民主传统
- 外部影响（俄罗斯、美国）的破坏

### 为什么吉尔吉斯斯坦与邻国不同？

**与哈萨克斯坦、乌兹别克斯坦、土库曼斯坦、塔吉克斯坦相比：**

- 安全机构规模较小
- 地区精英更自主
- 公民社会略微强大
- 没有权力至上的个人崇拜

**结果：** 革命是可能的——但持续的不稳定也是如此。[^12]

### 革命带来了民主吗？

**评价不一：**

**进步：**

- 2010年宪法：议会权力更强，总统权力更弱
- 中亚首次和平权力移交（奥通巴耶娃至阿坦巴耶夫，2011年）
- 一定程度的媒体自由得以保留

**损失：**

- 2010年族裔暴力暴露了国家软弱
- 后来的总统（阿坦巴耶夫、热恩别科夫）面临抗议
- 2020年：在有争议的选举后再次发生权力更迭
- 2021年：萨德尔·贾帕罗夫在冲进监狱后当选
- 民主巩固尚未完成[^13]

## 遗产

### 对吉尔吉斯斯坦而言

- 中亚政治上最具活力的国家
- 但也是最不稳定的国家
- 革命模式仍在继续
- 每一代人都充满希望，然后感到失望

### 对中亚而言

- 证明后苏联威权主义可以受到挑战
- 但也敲响了对后果的警钟
- 其他领导人（纳扎尔巴耶夫、卡里莫夫）从中吸取教训，避免了类似的命运
- 加强控制以防止“颜色革命”

### 对颜色革命理论而言

**吉尔吉斯斯坦的案例使简单的叙事复杂化：**

- 并非纯粹的民主运动
- 存在地区和部落因素
- 经济不满是核心
- 外部因素（俄罗斯、美国的影响）存在争议
- 革命并不保证民主[^14]
