## 概述

吉尔吉斯斯坦在后苏联历史中占有独特地位：它是中亚唯一一个通过民众起义推翻了两任总统的国家。2005年，郁金香革命推翻了自独立以来一直执政的阿斯卡尔·阿卡耶夫。2010年，在他继任的将近五年后，接替他的人——库尔曼别克·巴基耶夫——自己也被一场更为血腥的起义推翻。两次，愤怒的民众冲进政府大楼。两次，总统都逃离了国家。而两次，改革的承诺都未能实现。[^1]

吉尔吉斯斯坦的革命既展示了民众抗议的力量，也暴露了其局限性。人民表明他们不会容忍无限期的威权统治、舞弊的选举和家族腐败。但每一次革命之后都伴随着不稳定、族裔暴力，以及——最终——另一位令民主希望破灭的领导人。吉尔吉斯斯坦的反复革命揭示了一个在寻求真正民主但却难以建立可持续制度的国家。[^2]

## 第一部分：郁金香革命 (2005)

### 背景：阿卡耶夫的衰落

**阿斯卡尔·阿卡耶夫**曾被视为中亚最自由的领导人：

- 自1990年（独立前）起担任总统
- 物理学教授；最初以改革派形象示人
- 允许比邻国更多的言论自由
- 被称为中亚的“民主岛屿”

**但到了2005年：**

- 掌权15年
- 家族控制经济
- 镇压反对派
- 腐败盛行
- 民主的伪装荡然无存[^3]

### 导火索：欺诈性选举

**2005年2月-3月：** 议会选举

**问题：**

- 国际观察员发现存在重大舞弊
- 支持阿卡耶夫的候选人被宣布获胜
- 反对派候选人被取消资格或受到骚扰
- 抗议活动始于南部地区

**导火线：** 如果阿卡耶夫可以操纵议会，他就可以无限期地延长任期——甚至可能指定继承人（有人提及他的儿子或女儿）。[^4]

### 革命

**2005年3月：抗议活动蔓延**

**吉尔吉斯斯坦南部：**

- 贾拉拉巴德：反对派占领了政府大楼（3月20日）
- 奥什：发生类似占领事件
- 南部地区实际上已由反对派控制

**3月23日：**

- 反对派控制了全国三分之二的地区
- 阿卡耶夫仍掌控比什凯克

**3月24日：垮台**

**下午：**

- 15,000多名抗议者聚集在比什凯克阿拉套广场
- 游行至政府大楼
- 警方抵抗瓦解
- **抗议者冲进白宫（总统行政部门）**
- 随后发生抢劫

**阿卡耶夫逃亡：**

- 乘直升机前往哈萨克斯坦
- 后前往俄罗斯
- 再未返回

**伤亡：** 3人死亡（抢劫事件后），数十人受伤[^5]

### 后果

**临时政府：**

- **库尔曼别克·巴基耶夫：** 被任命为代总统
- **费利克斯·库洛夫：** 被释放出狱；担任安全主管
- 东北部与南部之间存在不安的联盟

**2005年4月4日：** 阿卡耶夫在莫斯科签署辞职书

**2005年7月10日：** 举行选举

- 巴基耶夫以88.9%的得票率获胜
- 最初承诺进行民主改革[^6]

## 第二部分：2010年革命

### 巴基耶夫的背叛

**期望与现实：**

| 承诺           | 现实                       |
| -------------- | -------------------------- |
| 民主改革       | 威权主义加剧               |
| 结束家族统治   | 巴基耶夫家族控制经济       |
| 清廉政府       | 大规模腐败                 |
| 新闻自由       | 媒体限制收紧               |

**2010年的主要问题：**

- 巴基耶夫之子马克西姆·巴基耶夫：被指控控制国家财政
- 能源和公用事业价格急剧上涨
- 政治对手被捕
- 公民社会受到限制

**2010年：** 与2005年前的民众不满几乎完全重演[^7]

### 导火索：经济困境

**2010年初：**

- 电价上涨170%
- 供暖成本翻倍
- 燃油价格上涨
- 许多人无力承担基本公用事业费用

**对于普通吉尔吉斯人来说，这是生死攸关的问题。** 表面上的原因是受益于巴基耶夫同伙的私有化交易。[^8]

### 革命

**2010年4月6日：塔拉斯**

- 抗议活动始于北部城镇塔拉斯
- 政府大楼被占领
- 州长一度被劫持
- 暴力事件蔓延

**2010年4月7日：比什凯克**

**上午：**

- 首都发生大规模抗议
- 数万人聚集
- 要求巴基耶夫辞职

**下午：**

- 部署了警察和安全部队
- 使用了催泪瓦斯、橡胶子弹
- **随后动用了实弹**
- 抗议者冲进国内安全总部
- 国家电视台被占领

**死亡人数：**

- 官方：87人死亡
- 一些估计：90+人
- **数百人受伤**

**比2005年更为血腥。**

**巴基耶夫逃亡：** 先是逃往吉尔吉斯斯坦南部，然后是哈萨克斯坦，最后是白俄罗斯[^9]

### 为什么2010年更血腥？

**主要区别：**

| 2005年                          | 2010年                            |
| ------------------------------- | --------------------------------- |
| 安全部队未开枪                  | 安全部队向抗议者开枪              |
| 阿卡耶夫迅速逃离                | 巴基耶夫进行抵抗                  |
| 3人死亡                         | 87+人死亡                         |
| 相对有序的权力过渡              | 随后陷入混乱                      |

**巴基耶夫从阿卡耶夫的垮台中吸取了教训——但应用了错误的教训。** 他认为武力会奏效。但事实并非如此。[^10]

### 后果与族裔暴力

**临时政府：**

- **罗莎·奥通巴耶娃：** 成为临时总统
- 中亚第一位女性国家元首
- 承诺真正的民主改革

**2010年6月：奥什暴力事件**

**灾难：**

- 吉尔吉斯斯坦南部吉尔吉斯族与乌兹别克族之间发生族裔冲突
- 470+人死亡（多为乌兹别克族）
- 400,000人流离失所
- 后苏联中亚地区最严重的族裔暴力事件

**原因争议：**

- 据称巴基耶夫的忠诚分子煽动
- 预存的族裔紧张关系
- 革命后的国家瓦解

**2010年革命的后果比前一次更为黑暗。**[^11]

## 历史评估

### 为什么会有两次革命？

**吉尔吉斯斯坦的模式：**

1. 反对派领导人通过抗议活动上台
2. 承诺改革
3. 巩固权力，充实家族
4. 压制反对派
5. 舞弊选举或经济危机引发起义
6. 循环往复

**根本问题：**

- 制度薄弱
- 地区（东北部-南部）分裂
- 基于恩庇的政治
- 没有强大的民主传统
- 外部影响（俄罗斯、美国）导致不稳定

### 为什么吉尔吉斯斯坦与邻国不同？

**与哈萨克斯坦、乌兹别克斯坦、土库曼斯坦、塔吉克斯坦相比：**

- 安全部门规模较小
- 地区精英更自治
- 公民社会稍强
- 没有无所不能的领袖崇拜

**结果：** 革命是可能的——但持续的不稳定也是如此。[^12]

### 革命带来了民主吗？

**结论不一：**

**成就：**

- 2010年宪法：议会权力加强，总统权力削弱
- 中亚首次和平权力交接（奥通巴耶娃到阿坦巴耶夫，2011年）
- 一定程度的新闻自由得以保留

**损失：**

- 2010年族裔暴力事件暴露了国家软弱
- 后来的总统（阿坦巴耶夫、热恩别科夫）也面临抗议
- 2020年：在有争议的选举后再次发生权力更迭
- 2021年：萨德尔·贾帕罗夫在冲进监狱后当选
- 民主巩固尚未完成[^13]

## 遗产

### 对吉尔吉斯斯坦

- 中亚最具政治活力的国家
- 但也是最不稳定的
- 革命模式仍在继续
- 每一代人充满希望，然后失望

### 对中亚

- 证明后苏联威权主义可以受到挑战
- 但也是对后果的警告
- 其他领导人（纳扎尔巴耶夫、卡里莫夫）进行了研究并避免了类似的命运
- 加强控制以防止“颜色革命”

### 对“颜色革命”理论

**吉尔吉斯斯坦的案例使简单的叙事复杂化：**

- 并非纯粹的民主运动
- 存在地区和氏族层面
- 经济不满是核心
- 外部因素（俄罗斯、美国的影响）存在争议
- 革命并不保证民主[^14]
