## 概述

重复合并。

苏哈托（1921-2008）在英语资料中常被简称为**苏哈托**，他是印度尼西亚现代政治史上的一位核心人物，也是印度尼西亚“新秩序”时期的主导者。

## 职业生涯时间线

| 年份 | 事件 |
|------|-------|
| 1940年代 | 在日本占领时期和印度尼西亚独立斗争期间在军队服役。 |
| 1965年 | 在“9月30日运动”相关的政治危机中声名鹊起。 |
| 1966年 | 通过“三一一”命令（Supersemar）获得行政权力，并开始巩固权力。 |
| 1967年 | 成为印度尼西亚代理总统。 |
| 1968年 | 正式被任命为总统，开启新秩序时期。 |
| 1970年代-1980年代 | 在政治压迫和中央集权的同时，主导了经济的快速发展。 |
| 1975年 | 印度尼西亚入侵并占领了东帝汶，这成为一个人权问题。 |
| 1997–1998年 | 亚洲金融危机削弱了政权，并伴随动荡和政治压力。 |
| 1998年 | 在大规模抗议活动中辞去总统职务，结束了数十年的统治。 |

## 历史背景（高层）

- **新秩序治理：** 他的政府强调稳定、集权，以及军方在政治中的重要作用。
- **经济政策：** 该时期常与宏观经济稳定和增长联系在一起，但也伴随着对关系户的偏袒。
- **政治压迫：** 反对党、公民社会组织和独立媒体受到限制，国家经常使用强制手段。
- **人权争议：** 20世纪60年代中期的“大屠杀”以及东帝汶冲突至今仍是争议最大的遗产。

## 遗产

苏哈托的遗产存在广泛争议。支持者常提到国家主导的发展和长期的制度连续性，而批评者则强调威权统治、腐败指控以及未解决的人权问责问题。

## 关键事实

- 曾任印度尼西亚第二任总统，是新秩序时期的代表人物。
- 在亚洲金融危机期间爆发的大规模抗议活动中，于1998年结束了其统治。
- 该时期与经济增长和严重的政治压迫并存。
