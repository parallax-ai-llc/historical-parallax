## 摘要

重复整合。

穆罕默德·苏哈托（1921-2008），在英文文献中通常被简称为**苏哈托**，是印度尼西亚现代政治史上的核心人物，也是印度尼西亚“新秩序”时期的主导者。

## 职业生涯时间线

| 年份 | 事件 |
|------|-------|
| 1940年代 | 在日本占领时期和印度尼西亚争取独立的斗争中服役于军队。 |
| 1965年 | 在“9月30日运动”的政治危机中声名鹊起。 |
| 1966年 | 通过《三一一军事命令》（Supersemar）获得行政权力，开始巩固权力。 |
| 1967年 | 成为印度尼西亚代理总统。 |
| 1968年 | 正式被任命为总统，开启“新秩序”时期。 |
| 1970年代–1980年代 | 在政治镇压和中央集权的同时，领导了经济的快速发展。 |
| 1975年 | 印度尼西亚入侵并占领了东帝汶，这成为一个重大的国际人权问题。 |
| 1997–1998年 | 亚洲金融危机削弱了政权，伴随着动荡和政治压力。 |
| 1998年 | 在大规模抗议活动中辞去总统职务，结束了数十年执政。 |

## 历史背景（高层概览）

- **新秩序治理：** 他的政府强调稳定、中央集权以及军队在政治中的重要作用。
- **经济政策：** 该时期常与宏观经济稳定和增长联系在一起，但也伴随着对关系密切的商业集团的偏袒。
- **政治镇压：** 反对党、公民社会行动者和独立媒体面临限制，国家经常使用强制性手段。
- **人权争议：** 1960年代中期的 মামলায়大规模暴力事件以及东帝汶的冲突仍然是最常被引用和讨论的遗产。

## 遗产

苏哈托的遗产备受争议。支持者常提到国家主导的发展和长期的制度连续性，而批评者则强调其专制统治、腐败指控以及未解决的人权问责问题。

## 关键事实

- 曾担任印度尼西亚第二任总统，是“新秩序”时期的代表人物。
- 其统治在亚洲金融危机期间的民众抗议浪潮中于1998年结束。
- 该时期既伴随着经济增长，也伴随着严重的政治镇压。
