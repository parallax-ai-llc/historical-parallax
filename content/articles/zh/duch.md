## 摘要

重复合并。参见“康克·埃弗”。

## 生平大事记

| 年份 | 事件 |
|------|-------|
| 1942 | 生于柬埔寨，原名康克·埃弗（Kaing Guek Eav）。[^1] |
| 1960年代 | 接受培训并担任教师（传记资料中常被提及为数学教师）。[^1] |
| 1970年代 | 在柬埔寨内战期间加入红色高棉运动。[^1] |
| 1975–1979 | 服务于民主柬埔寨政权；与内部安全机构相关联，并负责金边的S-21监狱（吐斯廉）的运作，在此期间大量被拘留者受到审讯并最终被杀害。[^1] |
| 1990年代 | 以化名生活；后被记者/援助工作者发现，引起公众对其角色的重新关注。[^1] |
| 2007 | 因在S-21犯下的罪行被柬埔寨当局逮捕。[^1] |
| 2010 | 被柬埔寨法院特别法庭（ECCC）判处犯有危害人类罪和战争罪；上诉后刑期被加重。[^1] |
| 2012 | ECCC确认最终判处无期徒刑。[^1] |
| 2020 | 服刑期间死亡。[^1] |

## 身份与姓名

- **俗称**: 杜奇 (Duch)
- **全名**: 康克·埃弗 (Kaing Guek Eav) (也有多种音译)
- **相关机构**: S-21（吐斯廉）安全监狱，金边

## 历史角色（概览）

杜奇最广为人知的是他在**S-21**监狱的领导角色。S-21是红色高棉安全机构在民主柬埔寨时期（1975–1979）的一个主要拘留和审讯中心。主要参考资料总结的记载表明，S-21是监狱被拘留者遭受审讯并随后被送往刑场的地方。[^1]

## S-21 / 吐斯廉（背景）

- 由于幸存的管理材料，S-21经常在历史和法律记录中被引用，成为被记录最好的红色高棉监狱之一。
- 杜奇的角色通常被描述为在国家安全框架内，**负责行政和监督**审讯、记录保存和监狱运作。[^1]

## 法律程序（ECCC）

杜奇由**柬埔寨法院特别法庭（ECCC）**起诉。ECCC是一个在联合国支持下设立的混合法庭，旨在处理红色高棉高层官员的罪行。

- 公开总结中提到的指控和定罪包括与S-21监狱运作相关的**危害人类罪**和**战争罪**。[^1]
- ECCC总结中的判刑历史包括一项初步判决和后来的**上诉结果，最终判处无期徒刑**。[^1]

## 附加背景（概览）

- **档案记录**: 吐斯廉/S-21经常在学术研究和法庭材料中被讨论，因为大量的管理文件得以保存，包括囚犯名单、审讯记录和照片。关于这一点的总结通常出现在对该监狱和ECCC案件材料的概述中。[^1]
- **在政权内的地位**: 杜奇通常被描述为红色高棉安全系统内的一名官员，而非战场指挥官；大多数公开记载都侧重于民主柬埔寨政权内部的拘留、审讯和官僚控制。[^1]
- **1979年后的公众知名度**: 生平简介通常指出，在民主柬埔寨政权垮台后的许多年里，他并未获得公众的广泛关注，后来在他与先前身份和角色的联系被报道后，才引起了更广泛的注意。[^1]
