## 概述

2018年11月17日，超过30万法国公民穿上了醒目的黄色安全背心——法律规定每辆法国汽车中必须配备的_gilets jaunes_——封锁了法国各地的道路、环形交叉路口和收费站。这场最初因反对燃油税上调而起的抗议，演变为法国自1968年以来最重大的民众起义。数月间，黄背心运动者占领街头，与防暴警察激烈冲突，迫使总统埃马纽埃尔·马克龙在其标志性经济改革上做出让步。[^1]

黄背心运动与此前的任何法国运动都截然不同。它没有领导者，没有政党，没有工会支持。参与者通过Facebook组织起来，他们来自法国精英阶层很少关注的地方：农村地区、小城镇、城市边缘地带——那些因缺乏公共交通而必须开车上班的地方。他们的愤怒表面上针对燃油税，但更根本的是针对一位被视为"富人总统"的领导人——一位前投资银行家，他废除了财富税却提高了柴油税。[^2]

马克龙最终做出了退让。燃油税被取消，最低工资被提高，约100亿欧元的让步措施相继出台。但伤痕犹在：11人在抗议中丧生，数千人受伤（包括被警察橡皮子弹击中导致的严重眼部伤害），法国社会的裂痕——巴黎与外省之间、全球化"赢家"与"输家"之间——赤裸裸地暴露在所有人面前。[^3]

## 历史背景

### 马克龙的法国

**埃马纽埃尔·马克龙：**

- 2017年当选（时年39岁）
- 前经济部长、投资银行家
- 中间派/自由主义改革议程
- 亲欧盟、亲市场

**他的改革：**

- 废除财富税（ISF）
- 改革劳动法（增加灵活性）
- 计划养老金改革
- 提高碳税

**公众印象：**

- "富人的总统"
- 脱离普通民众
- 态度傲慢

**到2018年底：** 支持率急剧下降。[^4]

### 燃油税

**核心问题：**

- 法国对燃油（尤其是柴油）课以重税
- 柴油：数十年来的首选燃料
- 2018年：柴油价格上涨16%
- 政府宣布2019年将进一步加税
- 被包装为气候政策（"碳税"）

**受影响群体：**

- 农村居民（没有公共交通）
- 郊区通勤者
- 工薪阶层
- 小企业主

**"我们已经入不敷出了。"**[^5]

## 运动经过

### 2018年11月17日：第一幕

**第一天：**

- 通过Facebook请愿发起
- 全国30万余人参与
- 道路被封锁
- 环形交叉路口被占领
- 发生部分冲突；1人死亡

**象征物：**

- 黄色安全背心（_gilet jaune_）
- 法律规定所有法国汽车必须配备
- 廉价、醒目、人人可得

### 每周"幕"行动（2018年11月 - 2019年3月）

**组织形式：**

- 每个周六 = 一"幕"
- 第一幕：11月17日
- 第二幕：11月24日
- 以此类推……

**巴黎成为战场：**

- 香榭丽舍大街爆发冲突
- 凯旋门遭到破坏
- 催泪弹、高压水炮
- 汽车被焚烧

**但同时：**

- 数以千计和平的环形路口占领
- 团结互助、交流对话
- "早起的法国"相遇了[^6]

### 2018年12月：高潮

**第三幕（12月1日）：**

- 最严重的暴力冲突
- 巴黎部分地区如同战场
- 135人受伤，412人被捕

**第四幕（12月8日）：**

- 政府出动89,000名警察
- 装甲车进入巴黎
- 有所暴力但程度减轻

**12月10日：**

- **马克龙发表全国讲话**
- 宣布让步措施
- 取消燃油税
- 最低工资每月增加100欧元
- 加班费免税
- 做出了（某种程度的）道歉

**但抗议仍在继续。**[^7]

### 2019年：衰退与延续

**2019年1月至3月：**

- 抗议持续但规模缩小
- 政府启动"大辩论"（全国协商）
- 马克龙巡访各地

**到2019年夏季：**

- 运动逐渐消退
- 核心诉求未获满足
- 但让步成果显著

**新冠疫情（2020年）：**

- 终结了每周六的常规抗议
- 运动宣布取得"象征性胜利"[^8]

## 死伤情况

**伤亡统计：**

| 类别                   | 数量         |
| ---------------------- | ------------ |
| 死亡                   | 11           |
| 重伤                   | 4,000+       |
| 眼部受伤（警方造成）   | 24+人失去眼睛 |
| 手部受伤（手榴弹造成） | 5人失去手掌   |
| 警察受伤               | 1,800+       |
| 被捕人数               | 12,000+      |

**死亡原因：**

- 封锁路段的交通事故
- 心脏病发作
- 冲突

**眼部伤害：**

- 法国警方使用"防御球发射器"（LBD）
- 橡皮子弹造成毁灭性伤害
- 成为警察暴力的象征
- 受到国际社会批评[^9]

## 后续影响

### 马克龙的让步

**他做出的让步：**

- 取消燃油税
- 最低工资每月增加100欧元
- 加班费免税
- 低收入退休人员税收减免
- 总计成本超过100亿欧元

**他未做出的让步：**

- 恢复财富税
- 根本性政策转变
- RIC（公民发起的公投）

### 政治影响

**对马克龙而言：**

- 挺过危机但元气大伤
- "大辩论"= 部分重启
- 随后又面临养老金抗议（2019-2020）
- 2022年连任（但对手是勒庞）

**对法国政治而言：**

- 黄背心未组建政党
- 部分人加入极左，部分人加入极右
- 玛丽娜·勒庞从中受益
- 对制度的深层不信任依然存在[^10]

## 历史评价

### 为什么会发生？

**直接原因：**

- 燃油税成为压垮骆驼的最后一根稻草
- 生活成本的挤压

**结构性原因：**

- 数十年的去工业化
- 城乡差距
- 全球化的失落者
- 政治阶层被视为脱离民众

**法国特殊因素：**

- 革命传统
- 对政府的不信任
- 中央集权国家 = 明确的抗议对象

### 为什么意义重大？

**一种新型运动：**

- 无领导者、无政党、无工会
- 通过社交媒体组织
- 具有强大的象征力量
- 将各种不满统一起来

**迫使政府退让：**

- 马克龙首次做出妥协
- "共和国前进"并非不可战胜

### 对比

| 运动               | 导火索       | 死亡人数 | 结果                     |
| -------------------- | ------------ | -------- | ------------------------ |
| 五月风暴（1968）     | 学生/工人    | 7        | 戴高乐受到动摇           |
| **黄背心运动（2018）** | **燃油税**   | **11**   | **税收取消；做出让步**   |
| 养老金抗议（2019）   | 改革         | 0        | 改革暂停（因新冠疫情）   |

**黄背心运动是自"六八年"以来规模最大的——而且他们部分地赢了。**[^11]

## 遗产

### 对法国

- 暴露了深层裂痕
- "边缘法国"获得了发声权
- 警察暴力引发辩论
- 但未形成持久的组织

### 对世界

- 启发了类似运动
- 黄背心被其他国家效仿（比利时、加拿大）
- 成为社交媒体动员的范本

### 留存的形象

**铭刻于记忆：**

- 香榭丽舍大街上的黄色海洋
- 催泪弹与街垒
- 环形路口上的团结

**"On Est Là"**（"我们在这里"）——黄背心运动的口号[^12]
