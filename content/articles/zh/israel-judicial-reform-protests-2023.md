## 概述

2023年大部分时间里，以色列经历了历史上规模最大、持续时间最长的抗议运动。每周六晚，数十万人聚集在特拉维夫和全国各地，反对政府计划中的司法改革。他们挥舞着以色列国旗，封锁高速公路，并警告说，总理本雅明·内塔尼亚胡正在摧毁民主本身。[^1]

拟议的改革将大幅削弱最高法院否决立法的权力，赋予政府对司法任命的控制权，并取消法院宣布政府“不合理”决定无效的能力。内塔尼亚胡和他的联盟——以色列历史上最右翼的联盟——认为法院的权力过大。批评者则看到了另一面：一位因腐败案受审的总理试图削弱唯一能追究其责任的机构。[^2]

抗议活动迫使内塔尼亚胡在2023年3月暂停了立法。然而，当他于7月强行通过“合理性”修正案时，抗议者们重新愤怒起来。接着，2023年10月7日，哈马斯袭击了以色列，造成1200人死亡。随着战争席卷全国，抗议运动在很大程度上暂停了。但潜在的危机并没有消失——它只是被推迟了。2024年1月，最高法院宣布“合理性”法律无效。关于以色列民主的斗争仍在继续。[^3]

## 历史背景

### 以色列的体系

**没有宪法：**

- 以色列有“基本法”
- 最高法院解释这些法律
- 可以否决议会（Knesset）的立法

**法院的权力：**

- 自20世纪90年代以来不断扩大（“宪法革命”）
- 可以宣布“不合理”的政府决定无效
- 司法任命：独立委员会
- 在右翼中存在争议

**内塔尼亚胡与法院：**

- 自2020年以来受审（腐败指控）
- 法院最终可能决定他的命运
- 批评者认为：改革是为了他的审判
- 支持者认为：改革早于审判[^4]

### 2022年大选

**2022年11月：**

- 内塔尼亚胡与极右翼联盟获胜
- 包括宗教犹太主义者和极端正统派政党
- 以色列历史上最右翼的政府

**联盟伙伴的议程：**

- 削弱法院（意识形态目标）
- 宗教豁免（极端正统派）
- 扩大定居点（极右翼）
- 内塔尼亚胡：避免入狱？

**司法部长 Yariv Levin：**

- 2023年1月宣布改革计划
- 长期支持司法改革的人[^5]

## 拟议改革

**计划内容：**

| 改革                              | 影响                                         |
| ----------------------------------- | -------------------------------------------- |
| 废除“合理性”条款                    | 法院不能宣布“不合理”的决定无效                  |
| 议会推翻权                          | 61票可推翻法院裁决                           |
| 司法任命                          | 政府控制任命                                 |
| 部长法律顾问                        | 不再提供有约束力的意见                       |

**批评者的看法：**

- 制衡机制的终结
- 威权主义倒退
- 类似于匈牙利和波兰
- 内塔尼亚胡在保护自己

**支持者的看法：**

- 法院权力过大
- 未经选举产生的法官不应凌驾于议会之上
- 是民主改革，而非破坏[^6]

## 抗议活动

### 第一阶段：运动兴起（2023年1月-3月）

**1月：**

- Levin宣布改革
- 首次抗议：数千人
- 每周六的示威活动开始

**2月：**

- 抗议规模扩大：8万+人
- 高科技行业组织起来
- 预备役人员警告：如果民主消亡，他们将不服役

**3月：**

- 抗议规模达到10万+人
- 威胁举行总罢工
- Histradut（工会联合会）呼吁罢工
- 经济部长警告经济可能崩溃

**2023年3月27日：**

- 内塔尼亚胡宣布“暂停”
- 表示将寻求对话
- 抗议者庆祝——但充满警惕[^7]

### 第二阶段：谈判破裂（2023年4月-6月）

**赫尔佐格总统的对话：**

- 试图促成妥协
- 反对派和执政联盟进行谈判
- 6月破裂

**原因：**

- 执政联盟不愿做出足够让步
- 反对派不接受任何版本
- 缺乏信任

**6月下旬，政府恢复了立法进程。**

### 第三阶段：“合理性”投票（2023年7月）

**2023年7月24日：**

- 议会通过“合理性”修正案
- 64票对0票（反对派抵制）

**抗议活动：**

- 大规模示威
- 预备役人员宣布：我们不会服役
- 包括飞行员、特种部队等
- 安全部门警告对国家造成的损害

**危机加剧——直到10月7日。**[^8]

### 2023年10月7日：一切发生变化

**哈马斯袭击：**

- 1200人死亡
- 250名人质被劫持
- 以色列历史上最严重的袭击
- 战争爆发

**抗议运动：**

- 暂停
- 国家紧急状态优先
- 一些组织者加入战争努力
- 对内塔尼亚胡的批评仍在继续（准备不足的失误）

**但宪法之战被搁置了。**

---

## 解决（部分）

### 2024年1月：最高法院裁定无效

**2024年1月1日：**

- 最高法院以8票对7票裁定
- “合理性”修正案违宪
- 最高法院首次推翻基本法修正案
- 内塔尼亚胡：“失望但会尊重”

**意味着什么：**

- 法院捍卫了自身独立性
- 改革在很大程度上陷入停滞
- 执政联盟没有足够票数推翻法院裁决
- 战争占据了政府的注意力[^9]

## 伤亡情况

**抗议活动显著非暴力：**

| 类别     | 人数                 |
| -------- | -------------------- |
| 死亡     | 0（直接与抗议相关）   |
| 受伤     | 极少                 |
| 逮捕     | 非常少               |

**警方使用武力：**

- 偶尔使用水炮
- 一些冲突
- 但是：总体上是和平的运动

**抗议者展现出的非凡纪律性得到了经常性的肯定。**

## 历史评估

### 为什么会发生？

**直接原因：**

- 内塔尼亚胡的法律困境
- 联盟伙伴的意识形态
- 强多数优势提供的机会

**根本原因：**

- 右翼长期以来对法院的 the grievances
- 文化分裂：世俗-自由派 vs. 宗教-民族主义者
- 对民主的定义存在争议

### 它表明了什么？

**以色列社会的分裂：**

- 特拉维夫 vs. 定居点
- 世俗 vs. 宗教
- 阿什肯纳兹-自由派 vs. 传统派
- “两个国家”的论调

**韧性：**

- 抗议规模巨大且持续
- 公民社会动员起来
- 预备役人员愿意拒绝服役

**局限性：**

- 最初未能阻止7月的法律通过
- 10月7日打断了一切
- 长期解决方案尚不明确

### 对比

| 运动        | 持续时间     | 死亡人数 | 结果                                       |
| --------------- | ------------ | -------- | ------------------------------------------ |
| **以色列 2023** | **9个月**    | **0**    | **最高法院裁定无效；改革停滞**                 |
| 匈牙利        | 零星出现     | 0        | Orbán 成功                               |
| 波兰        | 各式各样     | 0        | PiS 最终被选下台                           |

**以色列的抗议活动是任何民主国家人均抗议规模最大的之一。**[^10]

---

## 遗产

### 对以色列而言

- 揭示了深刻的分歧
- 法院（暂时）捍卫了独立性
- 10月7日的事件被掩盖，但并未解决
- 联盟（截至2024年底）仍在执政
- 问题被压制，但并未消亡

### 对民主而言

- 公民抵抗的典范
- 预备役人员的作用：独特
- 高科技行业作为政治参与者
- 但是：结局不确定

### 与10月7日的联系

**有人认为：**

- 国家分裂削弱了安全
- 政府因改革而分心
- 10月7日（部分）是其后果

**其他人则认为：**

- 是独立的问题
- 哈马斯独立选择了时机

**无论如何，10月7日冻结了政治斗争。**[^11]
