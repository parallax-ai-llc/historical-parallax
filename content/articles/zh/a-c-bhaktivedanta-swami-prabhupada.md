## 简介

A.C. 帕布帕德（A.C. Bhaktivedanta Swami Prabhupada）是一位印度灵性导师，创立了国际克里希纳意识协会（ISKCON），即广为人知的哈瑞·克里希纳运动。他在69岁时身无长物地前往美国，将克里希纳巴克提（奉爱）传播至全世界。他翻译并注疏了逾80卷吠陀文献，其中包括《原貌梵歌》（Bhagavad Gita As It Is）。ISKCON发展成为拥有数百万信徒的全球性运动。

## 职业生涯时间线

| 年份 | 事件 |
|------|-------|
| 1896 | 以阿巴伊·恰兰·德（Abhay Charan De）之名生于加尔各答 |
| 1922 | 与其灵性导师巴克提悉檀陀·萨罗斯瓦蒂（Bhaktisiddhanta Sarasvati）相遇 |
| 1959 | 接受桑雅萨（sannyasa，出家弃世） |
| 1965 | 69岁时抵达纽约市 |
| 1966 | 在纽约创立ISKCON |
| 1968 | 出版《原貌梵歌》（Bhagavad Gita As It Is） |
| 1975 | 创立帕布帕德图书基金会（Bhaktivedanta Book Trust） |
| 1977 | 在印度温达文（Vrindavan）辞世 |
