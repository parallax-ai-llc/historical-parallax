## 简介

格里戈里·叶菲莫维奇·拉斯普京是一位俄罗斯神秘主义者、自称的圣人，也是沙皇尼古拉二世宫廷中颇具影响力的人物。他出生在西伯利亚的一个农民家庭，凭借其声称的治疗能力和精神魅力，在圣彼得堡社会中获得了非凡的地位。在据称缓解了沙皇患有血友病的儿子阿列克谢皇太子的痛苦后，他与皇室家庭变得尤为亲近。他对皇后亚历山德拉的影响力，以及由此产生的对国家事务的影响力，使他备受争议，并成为罗马诺夫王朝批评的焦点。1916年12月，一群贵族认为他的存在损害了俄罗斯皇室，因此将他暗杀。

## 生平年表

| 年份 | 事件 |
|------|-------|
| 1869 | 1月21日出生于西伯利亚托博尔斯克省的波克罗夫斯科耶 |
| 约1897年 | 经历了一次精神转变，开始以朝圣者的身份游历 |
| 1903年 | 到达圣彼得堡；作为一名“斯塔列茨”（圣人）而闻名 |
| 1905年 | 通过黑山公主被引荐给皇室家庭 |
| 1907年 | 在一次血友病发作中首次被召来帮助阿列克谢皇太子；赢得了皇后亚历山德拉的信任 |
| 1908–1911年 | 在宫廷中的影响力日益增长；面临来自俄罗斯东正教会和政治精英的批评 |
| 1911年 | 首相彼得·斯托雷平试图将拉斯普京驱逐出圣彼得堡 |
| 1914年 | 躲过了僧侣伊利奥多尔的信徒希奥尼亚·古塞娃的暗杀企图 |
| 1915年 | 在第一次世界大战期间，沙皇尼古拉二世在前线时，拉斯普京对亚历山德拉和国家事务的影响力显著增强 |
| 1916年 | 12月30日，在圣彼得堡的尤苏波夫宫被费利克斯·尤苏波夫亲王及其同伙暗杀 |
