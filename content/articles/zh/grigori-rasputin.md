## 简介

格里戈里·叶菲莫维奇·拉斯普京是俄罗斯神秘主义者、自封的圣人，也是沙皇尼古拉二世宫廷中具有重大影响力的人物。他出生于西伯利亚一个农民家庭，凭借其所谓的治愈能力和精神魅力，在圣彼得堡上流社会中声名鹊起。据称他缓解了沙皇的血友病儿子——皇太子阿列克谢的痛苦后，与皇室家族的关系变得尤为亲近。他对皇后亚历山德拉以及由此对国家事务日益增长的影响力，使他备受争议，并成为对罗曼诺夫王朝批评的众矢之的。1916年12月，他被一群认为他的存在正在损害俄罗斯皇室的贵族所暗杀。

| 约1897 | 经历精神转变，开始作为朝圣者四处游历 |
| 1903 | 抵达圣彼得堡；以"斯塔列茨"（圣人）身份获得声誉 |

| 1907 | 首次被召唤协助皇太子阿列克谢度过血友病发作；获得皇后亚历山德拉的信任 |

| 1911 | 首相彼得·斯托雷平试图将拉斯普京驱逐出圣彼得堡 |
| 1914 | 在僧侣伊利奥多尔的追随者希奥尼娅·古塞娃发动的暗杀企图中幸存 |
| 1915 | 第一次世界大战期间沙皇尼古拉二世在前线时，拉斯普京对亚历山德拉和国家事务的影响力显著增加 |
| 1916 | 12月30日在圣彼得堡的尤苏波夫宫被费利克斯·尤苏波夫亲王及其同伙暗杀 |
