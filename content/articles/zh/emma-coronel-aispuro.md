## 概述

埃尔·查波的妻子，因承认协助运营锡那罗亚贩毒集团的毒品业务而认罪。

## 职业时间线

| 年份 | 事件 |
|------|------|
| 2007 | 与华金·"埃尔·查波"·古兹曼结婚（公开报道；后在美国法庭文件和媒体报道中被引用）。 |
| 2014 | 古兹曼被墨西哥当局逮捕；在随后的法律程序和媒体关注期间，科罗内尔仍是一位备受瞩目的公众人物。 |
| 2016 | 古兹曼在此前越狱后再次被捕；美国检察官后来指控科罗内尔在此期间协助进行通讯联络和运营支持。 |
| 2019 | 古兹曼在美国联邦法院被定罪并被判处终身监禁；科罗内尔继续出现在与该案件及其后续影响相关的报道中。 |
| 2021 | 因涉嫌协助锡那罗亚贩毒集团的相关指控在美国被捕；随后在联邦法院认罪。 |
| 2021 | 被判处数年美国联邦监禁及监督释放（据美国司法部和法院报道）。 |
| 2023 | 从美国联邦监管中释放，开始监督释放期（由主要新闻媒体报道；各报道细节有所不同）。 |

## 背景

- 科罗内尔·艾斯普罗是墨西哥公民，因与锡那罗亚贩毒集团前头目华金·"埃尔·查波"·古兹曼的婚姻而广为人知。
- 媒体报道常将她描述为前选美比赛参赛者；在古兹曼被捕、越狱及引渡相关报道之后，她的公众知名度显著提升。

## 指控与美国法律案件（概要）

- 美国检察官指控科罗内尔参与或协助了以下活动：
  - 在古兹曼被监禁期间为其传递信息，
  - 协助与古兹曼的通讯和行动相关的后勤工作，
  - 涉及其越狱及随后被再次逮捕期间的相关活动。
- 2021年，她在美国联邦法院就故意协助贩毒集团活动的指控认罪（据公开报道和官方公告摘要）。

## 重要意义

- 此案因将一位贩毒集团头目的知名配偶与涉嫌运营支持联系起来而引起广泛关注，说明了家庭成员和亲密伙伴如何可能被牵涉到有组织犯罪调查中。
- 该案也成为美国针对跨国犯罪组织支持网络的起诉策略的一个广泛报道的案例。
