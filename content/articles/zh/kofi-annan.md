## 摘要

占位符 — 外交官。

科菲·安南是一位加纳外交官，其职业生涯的大部分时间都在联合国度过，并成为联合国第七任秘书长（1997—2006年）。他广泛地被视为加强多边合作、推动联合国管理现代化，以及在维持和平、发展与人权领域推进全球规范的重要人物。

## 职业生涯年表

| 年份 | 事件 |
|------|-------|
| 1938 | 出生于黄金海岸库马西（今加纳）。 |
| 1950年代—1960年代 | 先后就读于曼萨皮姆中学、库马西科技学院、美国马卡莱斯特学院，以及日内瓦高级国际关系学院。 |
| 1962 | 加入联合国系统（世界卫生组织），开启了长达数十年的联合国职业生涯。 |
| 1990年代 | 担任联合国高级职务，包括在联合国外派任务扩展时期负责维持和平与人道主义行动。 |
| 1997 | 被任命为联合国第七任秘书长，是首位来自撒哈拉以南非洲的秘书长；开始第一任期。 |
| 2001 | 连任第二届秘书长；与联合国共同获得诺贝尔和平奖。 |
| 2006 | 结束秘书长任期；由潘基文接任。 |
| 2007 | 创立科菲·安南基金会（设于日内瓦），致力于和平、治理与可持续发展。 |
| 2012 | 担任联合国—阿拉伯国家联盟叙利亚问题联合特别代表（该任务于2012年晚些时候结束）。 |
| 2018 | 在瑞士伯尔尼逝世。 |

## 主要主题与贡献

- **联合国改革与管理：** 推动旨在改善各联合国机构之间协调、问责制及实地有效性的组织改革。
- **和平与安全：** 倡导强有力的维和任务授权，以及联合国、区域组织与主要大国之间更紧密的合作。
- **人权与平民保护：** 强调国际社会有责任保护平民免受大规模暴行侵害，并支持在联合国行动中加大对人权问题的关注。
- **发展与全球目标：** 支持由联合国主导的发展议程，以及横跨政府、公民社会和私营部门的合作伙伴关系。

## 背景（秘书长任期，1997—2006年）

- 在冷战后干预行动、主权与人道行动之间的争论以及大规模危机所塑造的时期，领导联合国前行。
- 就冲突预防、维持和平以及多边行动的合法性等议题，直接与各国领导人进行接触。
- 卸任后，通过科菲·安南基金会及其他国际倡议继续开展公共外交。

## 备注

- 在有关联合国不断演变的平民保护学说以及多边机构政治局限性的讨论中，安南经常被引用。
- 2001年获得的诺贝尔和平奖（与联合国共享）凸显了联合国在其任期内在国际合作与和平事业中所发挥的作用。
