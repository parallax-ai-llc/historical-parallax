## 简介

占位符 — 外交官。

科菲·安南是一位加纳外交官，他一生的大部分职业生涯都在联合国度过，并担任了第七任联合国秘书长（1997–2006年）。他与加强多边合作、革新联合国管理以及推进全球维和、发展和人权规范的努力紧密相连。

## 职业生涯时间线

| 年份 | 事件 |
|------|-------|
| 1938 | 出生于黄金海岸（今加纳）的库马西。 |
| 1950s–1960s | 就读于姆凡特皮姆学校、库马西科技大学、麦卡莱斯特学院（美国）和日内瓦研究生院。 |
| 1962 | 加入联合国系统（世界卫生组织），开启了长达数十年的联合国职业生涯。 |
| 1990s | 担任联合国高级职务，包括在联合国实地特派任务不断扩大的时期内负责维和及人道主义行动。 |
| 1997 | 被任命为联合国秘书长（第七任），成为首位来自撒哈拉以南非洲的联合国秘书长；开始第一个任期。 |
| 2001 | 连任联合国秘书长第二个任期；与联合国共同获得诺贝尔和平奖。 |
| 2006 | 结束联合国秘书长任期；由潘基文继任。 |
| 2007 | 创立科菲·安南基金会（日内瓦），专注于和平、治理和可持续发展。 |
| 2012 | 担任联合国-阿盟叙利亚联合特使（任期于2012年晚些时候结束）。 |
| 2018 | 在瑞士伯尔尼去世。 |

## 关键主题和贡献

- **联合国改革与管理：** 推动组织改革，旨在提高联合国各机构之间的协调性、问责制和外勤效率。
- **和平与安全：** 主张强有力的维和授权，以及联合国、区域组织和主要大国之间更紧密的合作。
- **人权与平民保护：** 强调国际社会保护平民免遭大规模暴行的责任，并支持在联合国行动中更加关注人权。
- **发展与全球目标：** 支持联合国牵头的发展议程以及跨越政府、民间社会和私营部门的伙伴关系。

## 背景（联合国秘书长，1997–2006年）

- 在冷战后干预、主权与人道主义行动的辩论以及大规模危机塑造的时期内领导联合国。
- 就预防冲突、维和以及多边行动的合法性等问题直接与国家元首进行接触。
- 卸任后，通过科菲·安南基金会及其他国际倡议继续进行公众外交。

## 注释

- 安南经常在关于联合国不断发展的平民保护原则以及多边机构的政治局限性的讨论中被提及。
- 2001年获得的诺贝尔和平奖（与联合国共享）突显了联合国在他任期内促进国际合作与和平努力的作用。
