## 简介

迪娜·阿舍尔-史密斯是第一位赢得世界短跑金牌（200米，2019年）的英国女子选手。

## 职业生涯时间线

| 年份 | 事件 |
|------|-------|
| 2011 | 开始参加国际比赛（青年级别）。 |
| 2013 | 赢得欧洲青年锦标赛冠军（包括200米）以及青年级别接力赛的成功。 |
| 2014 | 赢得世界青年锦标赛100米冠军。 |
| 2015 | 成为第一个在100米比赛中跑进11秒的英国女子选手；在世界田径锦标赛上打破英国200米纪录。 |
| 2016 | 赢得欧洲锦标赛200米冠军；同时也是英国队在里约奥运会4×100米接力赛中获得铜牌的成员。 |
| 2018 | 赢得欧洲锦标赛100米/200米双冠；还赢得了欧洲4×100米接力金牌。 |
| 2019 | 赢得世界田径锦标赛200米金牌（多哈），并在同届锦标赛中获得其他奖牌；创造了英国100米和200米纪录（10.83 / 21.88）。 |
| 2020 | 英国队在东京奥运会4×100米接力赛中获得铜牌的成员。 |
| 2022 | 赢得世界田径锦标赛200米铜牌（尤金）。 |
| 2024 | 英国队在巴黎奥运会4×100米接力赛中获得银牌的成员。 |

## 主要成就

- 第一位赢得世界田径锦标赛短跑金牌（200米，2019年）的英国女子选手。
- 保持着多项短跑项目（100米、200米和室内60米）的英国国家纪录。
- 欧洲100米和200米冠军，包括一次重要的锦标赛短跑双冠（2018年）。
- 钻石联赛100米冠军（2019年）。

## 比赛项目焦点

阿舍尔-史密斯主要是一位短跑运动员，擅长：

- 60米（室内）
- 100米
- 200米

## 个人最好成绩（精选）

（根据维基百科报道。）

- 100米：10.83秒（英国国家纪录，2019年）
- 200米：21.88秒（英国国家纪录，2019年）
- 60米（室内）：7.03秒（英国国家纪录，2023年）
