## 摘要

1975年4月17日，红色高棉部队进入金边。经历了多年的内战和美国轰炸的疲惫不堪的市民们，带着解脱的心情迎接了他们。战争结束了。和平已经降临。几个小时内，士兵们就开始用枪指着人们，强行疏散城市——每个医院、每个家庭。病患们被安置在病床上，在路上死去。两百万人被强行赶往农村。种族灭绝开始了。[^1]

在接下来的三年零八个月零二十天里，由波尔布特领导、奉行极端农民共产主义理想的红色高棉政权，杀害了约150万至200万柬埔寨人。这大约占总人口的四分之一。受害者因戴眼镜（知识分子的象征）、说外语、手部柔软或仅仅是来自城市而被处决。更多的人死于饥饿、疾病和田间的劳累。[^2]

红色高棉称之为“零年”。他们试图通过摧毁一切旧事物来建立一个新社会：教育、宗教、家庭、个人身份。柬埔寨的“杀人场”成为了意识形态疯狂的代名词——警示着乌托邦式的愿景与绝对权力结合的后果。该政权并非因为世界的干预而结束，而是因为越南的入侵和推翻。时至今日，许多肇事者仍未受到审判。[^3]

## 历史背景

### 1975年之前的柬埔寨

**王国时期：**

*   古高棉帝国（公元9-13世纪达到顶峰）建造了吴哥窟
*   法国保护国（1863-1953年）
*   西哈努克国王统治下的独立

**西哈努克的“中立”政策：**

*   试图让柬埔寨置身于越南战争之外
*   允许北越通过柬埔寨的补给线（胡志明小道）
*   在大国之间寻求平衡

**1970年政变：**

*   朗诺将军在西哈努克出国期间推翻了他
*   柬埔寨被卷入越南战争
*   西哈努克在流亡中与红色高棉结盟[^4]

### 红色高棉

**起源：**

*   1960年代成立的共产主义游击队
*   “红色高棉”（Khmer Rouge）——法语称谓
*   领导层：曾在法国受教育，受毛主义影响
*   波尔布特（萨洛特·萨尔）成为领导者

**意识形态：**

*   极端的农业共产主义
*   拒绝一切外国影响（包括越南共产主义）
*   回归神话般的农民纯洁
*   城市是腐败的象征；知识分子是敌人
*   不惜一切代价实现自给自足

**发展：**

*   1970年：几千名战士
*   1970-1975年：在美国的轰炸和内战的混乱中发展壮大
*   1975年：控制了农村；拥有7万名士兵[^5]

### 美国轰炸

1969年至1973年间，美国对柬埔寨进行了大规模轰炸：

*   **投下270万吨炸弹**——超过二战中同盟国所有炸弹的总和
*   表面上目标是北越的补给线
*   **造成3万至15万**平民死亡
*   摧毁了农村地区
*   将幸存者推向了红色高棉的怀抱
*   制造了红色高棉得以利用的混乱

**轰炸加剧了种族灭绝的条件。**[^6]

## 种族灭绝

### 第一天：1975年4月17日

**金边沦陷：**

*   红色高棉士兵身着黑衣进入首都
*   最初的庆祝——战争结束了
*   几小时内，士兵们下令疏散

**大疏散：**

> “你们必须离开。美国人要轰炸这座城市。只是三天。”

*   200万人被枪指着赶出家门
*   医院被清空；病患被推着病床离开
*   没有交通工具，没有食物，没有水
*   病弱者、老人和孩子在路上死去
*   任何反抗者都被枪杀
*   首都变成了一座鬼城

**这不是军事需要。这是意识形态：**城市是邪恶的；每个人都必须成为农民。[^7]

### “零年”

**红色高棉宣布彻底重塑文明：**

**废除：**

*   金钱（销毁货币）
*   私有财产
*   宗教（佛教僧侣被杀害或被迫劳动）
*   家庭结构（集体用餐，儿童分开）
*   教育（学校关闭）
*   市场、医院、法院
*   旧社会的任何残余

**新柬埔寨：**

*   “民主柬埔寨”（官方名称）
*   之前的一切都变得污秽、腐败
*   历史被抹去
*   外国大使馆关闭（除中国、朝鲜）
*   与世界完全隔绝[^8]

### 劳动营

**全体人口被迫从事农业劳动：**

**条件：**

*   每天工作12-16小时
*   食物不足（通常是稀米汤）
*   没有药物
*   疟疾、痢疾、精疲力竭
*   家庭分离
*   没有休息
*   口号：“留着你们无益，消灭你们无损”

**目标：**实现稻米产量过剩，以证明革命的成功

*   设定了不可能完成的配额
*   将歉收归咎于“敌人”
*   普遍存在饥荒

**因疾病、饥饿、劳累死亡估计人数：**75万-100万[^9]

### 处决

**除了劳动死亡外，红色高棉还系统性地处决了被视为敌人的人：**

**目标群体：**

*   前政府官员和士兵
*   受过教育的人（教师、医生、律师）
*   城市居民（被称为“新人”或“4月17日人民”）
*   少数民族（越南人、中国人、占族穆斯林）
*   宗教人士（佛教僧侣和尼姑）
*   任何有外国联系的人
*   任何表示异议的人

**如何识别：**

*   **戴眼镜** = 知识分子
*   **手部柔软** = 非农民
*   **说法语或英语** = 外国腐蚀
*   **会读书** = 潜在敌人

**方法：**

*   钝器（为节省子弹）
*   锄头、鹤嘴锄、竹竿、砍刀
*   用塑料袋闷死
*   将婴儿摔在树上
*   集体埋葬在坑中（通常是活埋）

**杀人场：**已发现超过2万个乱葬坑[^10]

### 吐斯廉（S-21）

**最臭名昭著的监狱：**

*   一所前高中被改造成酷刑/处决中心
*   1975-1979年：1.4万至1.7万人被监禁
*   **仅约12人幸存**（一些估计为7人）
*   由康克·吉·埃（“杜赫”）管理
*   保存详细记录：照片、酷刑下的供词
*   囚犯“供认”为中央情报局、克格勃或越南人工作
*   供认后：被带往（杀人场）春益（Choeung Ek）并被杀害

**照片：**数千张令人痛苦的嫌疑犯照片至今留存——死者的面容。[^11]

### 种族清洗

**特定群体成为目标：**

**越南人：**

*   红色高棉的意识形态是反越南的
*   柬埔寨的越南人死亡率高达100%
*   估计有10万多人被杀害或驱逐

**中国人：**

*   50%的华人被杀害
*   约21.5万人死亡

**占族（穆斯林）：**

*   36%-40%被杀害
*   清真寺被摧毁
*   被强迫吃猪肉；因拒绝而遭杀害
*   约10万人死亡

**泰国人、老挝人和其他少数民族：**遭受类似迫害

**这是政治性种族灭绝，也是族裔性种族灭绝。**[^12]

## 死亡人数

**红色高棉统治下死亡总人数：150万-200万**

| 原因     | 估计死亡人数 |
| -------- | ------------ |
| 处决     | 50万-100万   |
| 饥饿     | 50万-80万    |
| 疾病     | 20万-50万    |
| 劳累     | 上述已包含   |
| **总计** | **150万-200万** |

**这约占柬埔寨总人口的21%-25%。**

*   1975年人口：约700-800万
*   1979年人口：约500-600万

**四分之一的柬埔寨人死亡。**[^13]

## 结局

### 越南入侵

**该政权并非因内部抵抗而垮台，而是因为外国干预：**

**背景：**

*   红色高棉袭击越南边境地区
*   杀害越南平民
*   金边对河内越来越敌对
*   尽管双方都是共产主义者，但积怨已深

**1978年12月：**

*   越南发动全面入侵，投入15万军队

**1979年1月7日：**

*   越南军队占领金边
*   红色高棉逃往泰国边境
*   种族灭绝结束——历时3年零8个月零20天

**越南占领一直持续到1989年。**[^14]

### 余波

**国际反应：**

*   中国（红色高棉的盟友）入侵越南；爆发短暂边境战争
*   美国和西方国家在1990年之前继续承认红色高棉控制的联合国席位
*   冷战政治压倒了人权
*   红色高棉游击队继续战斗直到1999年

**柬埔寨：**

*   洪森（前红色高棉变节者）被越南扶植上台
*   统治柬埔寨直到2023年
*   威权稳定取代了大规模屠杀
*   重建缓慢而痛苦[^15]

## 正义

### 柬埔寨特别法庭（ECCC）

**成立：**2003年（协议）；2007年开始审判

**任务：**审判红色高棉高级领导人的罪行

**审判：**
| 案件   | 被告           | 结果               |
| ------ | -------------- | ------------------ |
| 001号案 | 杜赫           | 无期徒刑（2012年） |
| 002号案 | 农谢，乔森潘   | 均判无期徒刑       |
| 003-004号案 | 暂停或驳回     |                    |

**局限性：**

*   仅审判了少数几人
*   审判过程缓慢且昂贵
*   大多数肇事者得以逍遥法外
*   波尔布特从未受审
*   未追究低级干部责任

**审判开始时，许多关键人物已经去世或濒临死亡。**[^16]

## 历史评估

### 为什么会发生？

**多种因素：**

1.  **美国轰炸：**摧毁了柬埔寨农村；使人口激进化
2.  **冷战：**柬埔寨夹在大国之间
3.  **极端意识形态：**毛主义的逻辑推演
4.  **孤立：**政权不受外部制约
5.  **农民怨恨：**真实的诉求被利用
6.  **绝对权力：**没有制约领导层的机构
7.  **非人化：**敌人被视为需要“扫除”的“微生物”

### 与其他种族灭绝的比较

| 特征           | 红色高棉           | 大屠杀            | 卢旺达            |
| -------------- | ------------------ | ----------------- | ----------------- |
| 死亡比例       | 约25%              | 约60%犹太人       | 约75%图西族      |
| 主要武器       | 饥饿，简易工具     | 工业化杀戮        | 砍刀              |
| 意识形态       | 农业共产主义       | 种族纯洁          | 族裔民族主义      |
| 持续时间       | 3年以上            | 约4年             | 100天             |
| 外部干预？     | 越南结束了它       | 同盟国胜利        | 极少              |

### 独特性

*   **自我灭绝：**国家杀害自己的国民，而非外国群体
*   **阶级即种族：**城市居民被视为一种族裔类别
*   **整体性：**试图从零开始重建文明
*   **有罪不罚：**许多肇事者融入社会；起诉极少[^17]

## 遗产

### 对柬埔寨而言

**持续的创伤：**

*   幸存者平均年龄现已60-70多岁；记忆正在消退
*   年轻一代对细节往往不了解
*   心理健康负担
*   缺失的一代（受过教育者被杀害）
*   杀人场仍在被发现
*   幸存者和肇事者并肩生活

**纪念：**

*   春益（杀人场）纪念碑
*   吐斯廉博物馆
*   国家纪念日（5月20日）
*   口述历史项目[^18]

### 对世界而言

**教训：**

*   不受约束的意识形态可以为任何事情辩护
*   冷战政治可以忽视种族灭绝
*   “永不再发生”必须付诸行动
*   迟到的正义并非正义
*   从大规模暴行中重建需要几代人

**法律先例：**

*   ECCC为国际刑法做出了贡献
*   在柬埔寨背景下定义了反人类罪
*   但也显示了迟到正义的局限性[^19]

### 文化影响

**《杀人场》（1984年）：**

*   奥斯卡获奖影片
*   向世界展示了柬埔寨的恐怖
*   幸存者约翰·海·吴（Haing S. Ngor）获得最佳男配角奖
*   他于1996年在洛杉矶被谋杀

**文献记录：**

*   DC-Cam（柬埔寨文献中心）保存证据
*   记录受害者证词
*   学校的教育项目[^20]
