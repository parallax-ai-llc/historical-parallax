## 摘要

挽颂·巴博拉·帕里巴特拉亲王，那空沙旺二世亲王（1904-1959），是一位泰国皇室成员，他于1952年将自己的私人住所改建为素万帕卡德宫博物馆，成为第一位向公众开放家园的泰国皇室成员，从而改变了泰国的文化保护方式。作为显赫的帕里巴特拉·素坤婉·帕里巴特拉亲王唯一的在世儿子，以及朱拉隆功国王（拉玛五世）的孙子，他见证了1932年君主专制制的动荡终结及其后的流亡。他最伟大的遗产不在于政治权力，而在于其通过博物馆发展和考古保护，使泰国文化遗产的获取民主化。[^1][^2]

---

## 早年生活与背景

挽颂·巴博拉·帕里巴特拉亲王于1904年12月5日出生于泰国最具影响力的皇室家族之一。他的父亲帕里巴特拉·素坤婉·帕里巴特拉亲王是朱拉隆功国王（拉玛五世）与苏坤拉·玛拉诗丽王后所生的第33个孩子、第13个儿子，这使得挽颂亲王成为泰国最受尊敬的君主之一的直系孙辈。[^3] 作为他父亲的王后所生的八个孩子中唯一活到成年的儿子，挽颂亲王自出生起就肩负着重要的王朝责任。[^4]

他的童年受到了他父亲非凡政治生涯的影响，其父在20世纪初曾担任海军元帅、泰国皇家陆军总参谋长、泰国皇家海军司令，并担任过多个内阁职位。[^5] 根据1924年的《继承法》，年幼的挽颂亲王作为苏坤拉·玛拉诗丽王后唯一在世的男性后裔，以及已故普密蓬·阿杜德国王的远房表亲，拥有成为泰国王位继承人的潜在身份。[^6]

1924年的《继承法》本身就带有对他家族的苦涩讽刺——哇集拉瓦国王（拉玛六世）起草该法的部分原因是为了破坏帕里巴特拉亲王的继承希望，因为国王鄙视他，转而提升他自己的同父同母兄弟。[^7] 这种政治上的边缘化预示着家族动荡的未来。在父亲强大的政治影响阴影下成长，挽颂亲王在泰国从君主专制向君主立宪制过渡的时期，经历了皇室血统的特权与危险。[^8]

## 权力崛起

与他父亲走向军事和政治主导地位的轨迹不同，挽颂亲王的“崛起”遵循了一条不同的道路——一条文化而非政治影响力的道路。定义他角色的关键时刻不是通过任命或征服，而是通过危机和流离失所。

1932年6月24日，人民党（Khana Ratsadon）发动了一场不流血的政变，结束了泰国长达150年的扎克里王朝绝对统治。[^9] 当时担任国王普密蓬·阿杜德（拉玛七世）休假期间的摄政的帕里巴特拉亲王，于当天早上被捕。[^10] 革命者认为他过于强大和危险，无法释放，最终迫使他流亡。帕里巴特拉亲王前往爪哇，再未返回，于1944年在日本控制的万隆去世。[^11]

对于28岁的挽颂亲王来说，他当时已与拉差翁·潘提普·德瓦库拉结婚，并育有两岁的玛西公主，这场革命粉碎了他所有剩余的政治野心。[^12] 家族随帕里巴特拉亲王一同流亡，先是去了爪哇，然后是英国，玛西公主在那里上了小学。[^13] 在新的宪政秩序下，皇室亲王被明确排除在国家委员会和议会之外。[^14]

当家族最终返回泰国时，挽颂亲王面对的是一个已经改变的王国，他的皇室身份拥有社会声望，但没有政治权力。他没有退隐到无关紧要的境地，而是发现了一种不同的影响力——文化领导力。他与潘提普·德瓦库拉的婚姻（潘提普于1909年3月8日出生于她祖父宾提查莱姆·邦纳的住所）被证明是至关重要的。[^15] 在20世纪40年代，他们开始共同收集非凡的泰国古代文物和传统建筑，为他们即将迎来的决定性成就做准备。[^16]

## 主要成就

挽颂亲王最具变革性的成就在1952年，他与潘提普亲王妃将他们在曼谷的私人住所改建为素万帕卡德宫博物馆，成为第一批向公众开放其家园和收藏品的泰国皇室成员。[^17] “素万帕卡德”的名字意为“卷心菜地”，指的是这片土地曾经拥有的菜园。[^18] 这一开创性的举动在泰国迅速现代化、传统文化面临被抹去的关键时刻，使泰国文化遗产的获取民主化。

博物馆的建筑基础由四座传统的泰国柚木房屋构成，可追溯到18至19世纪，这些房屋曾是亲王家族的财产，他精心从清迈迁来，并在宫殿内重建。[^19] 这些房屋以经典的泰国住宅风格建造，建在高高的柱子上，离地面两米高，并通过优雅的抬高走道连接，形成了一个和谐的建筑群，保存了正在消失的建筑传统。[^20] 每座房屋都成为不同收藏品的画廊，涵盖了史前文物到19世纪的珍宝。

班清陶器收藏代表了素万帕卡德宫最重要的考古藏品之一。1966年，当美国学生斯蒂芬·杨在乌隆他尼附近被一个陶罐绊倒时，他将其呈给了潘提普亲王妃，后者立即认识到其重要性，并通知了泰国艺术部和福特基金会的伊丽莎白·莱昂斯。[^21] 潘提普亲王妃组织了一次前往班清的考察，收集陶片进行测年，并努力保护这些文物免遭附近乌隆基地驻扎的美国士兵的搜集。[^22] 该收藏品现在包括公元前1666年至公元前500年的陶器和青铜首饰，代表了东南亚最重要的史前文化之一。[^23]

也许这座宫殿的皇冠明珠是漆盒阁，一座17世纪的建筑，最初是阿瑜陀耶佛教寺庙的图书馆。[^24] 挽颂亲王在班林寺发现其破败不堪，于1959年获得了住持的祝福将其搬迁。[^25] 漆盒阁被精心拆卸、运往曼谷，并被细致地修复，作为给潘提普亲王妃的生日礼物。[^26] 其精致的黑色背景上的金色漆艺描绘着佛教故事和泰国神话，使其成为泰国最重要的艺术瑰宝之一。

宫殿还收藏有高棉雕塑，包括潘提普亲王妃在柬埔寨边境附近发现的7世纪的乌玛雕像躯干，来自宋、元、明时期的中国陶瓷，以及素可泰时期的泰国素万那普青瓷。[^27] 挽颂亲王和潘提普亲王妃于1952年建造了一个接待阁，用于展示这些古董，每周开放几天供游客参观，并使用门票收入来推广泰国文化。[^28]

## 争议与批评

尽管挽颂亲王的文化保护工作赢得了广泛赞誉，但他的地位并非没有复杂性和批评。最显著的争议涉及他家族的政治边缘化以及1932年革命的背景。

一些历史学家认为对帕里巴特拉家族的待遇是不公正地迫害了有能力的管理者。帕里巴特拉亲王在军事和文职领域为泰国做出了杰出贡献，但人民党认为他“过于强大”并迫使他流亡。[^29] 挽颂亲王随后退出政治生活，可以被解释为审慎的适应，也可以被视为对公共服务的损失——他潜在的治理贡献永远未被探索。

素万帕卡德宫古董的收购也引起了审查。潘提普亲王妃“经常被指责使收藏变得时髦”，尤其是在班清陶器方面。[^30] 她声称收集这些标本是为了将它们从附近的美国空军基地驻扎的士兵手中解救出来，批评者认为皇室收藏——即使是出于保护意图——助长了考古材料的商品化，并鼓励了盗窃。保护与挪用之间的界限仍然在伦理上复杂。

1924年的《继承法》将挽颂亲王指定为潜在继承人，该法本身也存在争议，由哇集拉瓦国王起草，部分原因是“出于恶意，以破坏他同父异母兄弟帕里巴特拉亲王的希望”。[^31] 这份文件将挽颂亲王置于一个旨在边缘化他父亲家族的继承体系中，引发了对基于个人敌意而非功绩或明确原则的皇室等级制度合法性的质疑。

此外，挽颂-潘提普基金会后来在曼谷唐人街管理房产也招致批评。基金会终止了长期租赁，转为年租协议，给寻求保护文化遗产的当地社区带来了不确定性——这与挽颂亲王自身的保护使命形成了讽刺的对比。[^32]

最后，一些现代学者质疑以皇室为中心的博物馆叙事是否充分代表了泰国多元的文化。通过过度关注宫廷艺术和皇室收藏，素万帕卡德宫等机构可能无意中强化了精英阶层的观点，同时边缘化了普通泰国人民的文化贡献。[^33]

## 个人生活

挽颂亲王与拉差翁·潘提普·德瓦库拉结婚，结成了将定义他们一生的伙伴关系。[^34] 潘提普公主于1909年3月8日出生，出身于一个有影响力的贵族家庭，是宾提查莱姆·邦纳的后裔。[^35] 他们的婚姻育有一女，玛西·素坤婉·帕里巴特拉公主，出生于1930年8月25日，她后来成为一位有成就的艺术家。[^36]

1932年的革命极大地扰乱了他们的个人生活。在暹罗革命后，挽颂亲王离开泰国，玛西公主的童年大部分时间都在印度尼西亚的爪哇和英国度过。[^37] 尽管这一流亡时期是出于政治动机，但它让这个家庭接触到了国际视角，这些视角可能影响了他们后来的文化工作。流离失所的经历可能加深了挽颂亲王对通过物质文化保护泰国身份的欣赏。

返回泰国后，这对夫妇在曼谷的帕亚泰路上建立了他们的住所，这块土地以前是卷心菜地。[^38] 他们的家成为了艺术家、学者和文化爱好者的聚会场所。潘提普公主在博物馆的发展中也发挥了同样重要的作用，亲自发现了重要的考古文物并管理着藏品。[^39]

挽颂亲王于1959年9月15日在曼谷去世，享年54岁。[^40] 在现有历史资料中没有记录他具体的死因。他的去世发生在完成漆盒阁修复作为送给妻子的生日礼物仅仅几个月后，这个最后的保护项目成为了他们合作的凄美象征。[^41]

潘提普公主比他多活了28年，继续他们的文化使命。她创立了挽颂-潘提普基金会以支持艺术和文化，于1961年将素万帕卡德宫的部分空间改建成私人艺术画廊，并于1974年在成立比拉西当代艺术学院中发挥了领导作用。[^42] 她于1987年去世，对泰国的当代艺术场景产生了深远影响。[^43]

他们的女儿玛西公主成为了一位杰出的艺术家，直到2013年7月9日去世，都延续着父母的文化遗产。[^44] 这个家族三代人对泰国艺术的奉献代表了一个非凡的文化赞助王朝。

## 遗产

挽颂亲王的遗产通过建立皇家财产向公众开放博物馆的模式，从根本上重塑了泰国文化保护。素万帕卡德宫至今仍是曼谷最重要的文化机构之一，在1952年开馆七十多年后，继续向游客传授泰国文化遗产。[^45] 该博物馆的成功启发了其他保护项目，并表明泰国贵族即使在失去政治权力后，也能为国家的文化生活做出有意义的贡献。

他将门票收入用于“推广泰国文化”的愿景，创造了一种可持续的资金模式，使文化机构在保持对普通泰国人开放的同时能够自给自足。[^46] 这代表了文化获取方式的根本性民主化——在此之前，这样的宝藏仍被锁在私人皇室庭院里。

漆盒阁的抢救确立了建筑保护的重要先例。在泰国迅速现代化并淘汰传统建筑的时候，挽颂亲王表明，旧建筑可以被搬迁和修复，而不是被摧毁。[^47] 这种方法影响了泰国 subsequent 的保护工作。

班清收藏的意义超出了素万帕卡德宫。潘提普亲王妃在帮助识别和保护班清文物方面的初步工作，为该遗址最终于1992年被列入联合国教科文组织世界文化遗产做出了贡献。[^48] 素万帕卡德宫的陶器收藏为理解东南亚史前文明提供了重要的比较材料。

挽颂-潘提普基金会继续这对夫妇的慈善事业，尽管在财产管理方面存在争议。[^49] 该基金会尤其通过比拉西当代艺术学院对泰国艺术的支持，通过为非学院派艺术家提供展览空间，帮助发展了泰国的当代艺术场景。[^50]

挽颂亲王的一生展示了流离失所的皇室成员的另一条道路——他没有执着于失去的政治权力或退隐到无关紧要的境地，而是将贵族资源投入到文化保护中。这种模式影响了其他皇室成员在君主立宪制下的泰国的角色构想。

## 历史评估

### 积极评价

泰国文化史学者认为挽颂亲王是一位富有远见的保护主义者，他在积极的现代化时代认识到了传统文化的价值。艺术史学家尤其赞赏他的建筑保护工作，指出素万帕卡德宫的传统泰国房屋是仍在向公众开放的国内建筑的最佳范例之一。[^51] 他开创的搬迁和修复方法已被研究并应用于后续的保护项目。

博物馆专业人士称赞挽颂亲王和潘提普亲王妃使泰国的文化获取民主化。他们将收藏品“每周开放几天”，并利用门票收入进行文化推广，创建了一个可持续的公共机构，而不是一个虚荣项目。[^52] 这种方法比现代博物馆管理原则早了几十年。

考古学学者承认潘提普亲王妃在保护班清材料方面发挥的关键作用，尽管她的收集方法面临批评。她对斯蒂芬·杨的陶器发现的即时识别以及随后保护和研究文物的努力，有助于确立班清对世界史前史的重要性。[^53] 没有素万帕卡德宫的收藏，对班清材料的比较分析将大大贫乏。

文化史学家认为挽颂亲王的成就代表了皇室对君主立宪制的成功适应。他没有反对新的政治秩序或沉溺于反动的怀旧，而是利用自己的资源和地位，找到了为泰国社会做出贡献的富有成效的方式。[^54] 这种务实的方法帮助稳定了泰国从君主专制向君主立宪制的过渡。

泰国的当代博物馆管理者将素万帕卡德宫视为一个鼓舞人心的榜样，表明私人收藏品在保持家庭管理的同时，可以服务于公众教育。[^55] 该宫殿目前仍由家族管理，现通过玛西基金会运营，这表明了挽颂亲王愿景的可持续性。

### 消极评价

批评学者质疑皇室博物馆叙事是否充分服务于泰国各民族。通过过度关注宫廷艺术、皇室收藏和贵族品味，素万帕卡德宫可能强化了精英的观点，同时边缘化了普通民众的文化表达。[^56] 该博物馆展示了“高雅文化”，同时可能贬低了民间传统。

一些历史学家认为挽颂亲王退出政治参与是一个错失的机会。作为一位受过良好教育的皇室成员，根据1924年的继承法，他可能在泰国的宪政发展中扮演建设性角色，如果他选择政治参与而非文化工作的话。[^57] 他的退出，尽管鉴于他父亲的流亡是可以理解的，却代表了有能力的领导力的缺失。

考古伦理学者批评了挽颂亲王和潘提普亲王妃所代表的私人收藏文化，即使是出于保护意图。他们对班清材料、高棉雕塑和其他古董的收购，是在缺乏专业考古学所要求的系统性记录和背景记录的情况下进行的。[^58] 这种由皇室参与“制造时髦”的收藏文化，可能鼓励了盗贼和商人。

政治史学家注意到帕里巴特拉家族在1932年革命中的复杂遗产。尽管帕里巴特拉亲王被迫流亡，但关于他实际威胁宪政秩序程度与革命者偏执的质疑仍然存在。[^59] 挽颂亲王对这些政治问题的沉默——尽管出于个人安全可以理解——使得历史上的模糊之处未能解决。

当代社区倡导者批评挽颂-潘提普基金会在曼谷唐人街等地区的房产管理，那里的租赁终止给当地文化保护工作带来了不确定性。[^60] 这与挽颂亲王自身的保护使命形成了讽刺的矛盾，并对机构偏离创始人意图提出了质疑。

| 1930 | 玛西·帕里巴特拉公主于8月25日出生[^64] |
| 1932 | 暹罗革命结束君主专制；父亲帕里巴特拉亲王于6月24日被捕并流亡[^65] |
| 1932-1940s | 家族流亡于印度尼西亚爪哇和英国[^66] |
| 1940s | 返回泰国；与潘提普公主开始收集古董[^67] |
| 1944 | 父亲帕里巴特拉亲王在印度尼西亚万隆流亡中去世[^68] |
| 1952 | 向公众开放素万帕卡德宫博物馆，成为第一位这样做的泰国皇室成员[^69] |
| 1959 | 将漆盒阁从阿瑜陀耶搬迁并修复，作为送给潘提普公主的生日礼物[^70] |
| 1959 | 9月15日于曼谷去世，享年54岁[^71] |
| 1961 | 潘提普公主将宫殿空间改建为私人艺术画廊（遗赠影响）[^72] |
| 1966 | 潘提普公主帮助识别和保护班清陶器发现（遗赠影响）[^73] |
| 1974 | 在潘提普公主的领导下成立比拉西当代艺术学院（遗赠影响）[^74] |
| 1987 | 潘提普公主去世，她继续了他的文化使命28年[^75] |
| 1992 | 班清被指定为联合国教科文组织世界文化遗产，部分归功于素万帕卡德宫的收藏[^76] |
| 2013 | 女儿玛西公主于7月9日去世，结束了三代人的文化遗产[^77] |

## 名言

> “他们的珍贵藏品构成了泰国遗产的一部分，观看它们应该让每个人都能接触到。”
>
> — 挽颂亲王和潘提普亲王妃在开放素万帕卡德宫时的理念，由博物馆历史学家转述，他们记录了他们的信念，即皇室收藏应服务于公众教育，而不是保留为私人宝藏。[^78]

> “推广泰国文化”
>
> — 挽颂亲王和潘提普亲王妃声称使用素万帕卡德宫门票收入的目的，为文化机构的资金建立了一个可持续的模式，同时保持了可及性。[^79]

> “我将尽量多地保存样本”
>
> — 潘提普亲王妃在开始收集班清陶器时的声明，作为对附近美国空军基地搜集纪念品者的回应，正如考古学史对该遗址发现的记载。[^80]

*注意：挽颂亲王本人的直接引语在现有历史资料中并不广为记载。他的遗产主要体现在他的行动和创造的机构中，而不是通过书面或录音讲话。上述引语代表了历史学家和博物馆记录中记载的报告陈述和哲学立场。*
