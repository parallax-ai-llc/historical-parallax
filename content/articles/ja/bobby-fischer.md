## 経歴年表

| 年 | 出来事 |
|------|-------|
| 1956 | 13歳で全米ジュニアチェス選手権を制覇。ドナルド・バーンとの「世紀の一局」が国際的な注目を集める。 |
| 1957–1958 | 全米選手権で優勝（8度の全米タイトル獲得の第1回目）。 |
| 1958 | インターゾーナル経由で資格を取得し、グランドマスターの称号を受ける（当時の最年少記録）。 |
| 1962 | ストックホルム・インターゾーナルで優勝。その後、数年間にわたりトーナメント参加を減らす。 |
| 1970–1971 | トップレベルへ復帰。インターゾーナルおよび候補者マッチに勝利。マルク・タイマノフとベント・ラーセンに対して2度の6-0圧勝を記録した後、ティグラン・ペトロシアンを下す。 |
| 1972 | レイキャビクでボリス・スパスキーを破り、世界チェスチャンピオンに輝く。 |
| 1975 | 対局条件をめぐる争いからアナトリー・カルポフへのタイトル防衛を行わず、タイトルを剥奪される。 |
| 1992 | ユーゴスラビアでスパスキーとの非公式リマッチを行い、米国の制裁に違反。法的措置を招く。 |
| 2004 | パスポート問題で日本にて拘束。後にアイスランド国籍を付与される。 |
| 2008 | アイスランドのレイキャビクで死去。 |

## 棋風と貢献

- 深い序盤の準備と、局面の客観的要求に応じたシャープで不均衡なポジションの追求で知られる。
- 米国のエリート棋士における研鑽の水準を専門化することに貢献し、解析とエンドゲーム技術の研究を強く重視した。
- 白番でのキングズ・インディアン・アタックや、黒番でのシシリアン・ディフェンスによる積極的な戦いが広く語られるが、レパートリーは幅広かった。

## 公的論争（概要）

- 対局条件、スケジュール、賞金などをめぐり、主催者やチェス当局としばしば衝突した。
- 晩年の公的発言は広く批判を受け、チェスの実績とは別に彼の公的な遺産の大きな部分を占めている。

## 遺産

- 20世紀最強の棋士の一人として広く挙げられる。1971年候補者マッチにおける圧倒的な勝利の連続はその支配力の象徴として頻繁に取り上げられる。
- 1972年の世界選手権マッチは、米国およびそれ以外の地域でチェスへの関心を大きく高めるきっかけとなった。
- 彼の棋譜は戦術的計算、実践的エンドゲームの指し回し、小さなアドバンテージの活かし方として広く研究されている。
