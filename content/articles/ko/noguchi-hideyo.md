## 요약

노구치 히데요는 매독, 황열병을 비롯한 여러 감염병 연구에 중요한 기여를 한 일본의 저명한 세균학자였다. 뉴욕의 록펠러 연구소에서 근무하며 마비성 치매 환자의 뇌에서 매독 스피로헤타를 입증해 보였다. 서아프리카에서 황열병을 연구하던 중 자신도 황열병에 걸려 사망했다.

## 경력 연표

| 연도 | 사건 |
|------|-------|
| 1876 | 일본 후쿠시마현 이나와시로에서 출생 |
| 1897 | 도쿄의 의학교 졸업 |
| 1900 | 미국으로 건너감 |
| 1904 | 록펠러 연구소에 합류 |
| 1911 | 매독 스피로헤타 배양에 성공 |
| 1913 | 마비성 치매 환자의 뇌에서 트레포네마 팔리듐 발견 |
| 1918 | 황열병 및 기타 열대 질환 연구 |
| 1927 | 황열병 연구를 위해 서아프리카로 떠남 |
| 1928 | 골드코스트 아크라에서 황열병으로 사망 |
