## Resumo

Brendan Eich é um tecnologista americano que criou a linguagem de programação JavaScript em 1995 enquanto trabalhava na Netscape. O JavaScript se tornou a linguagem de programação mais amplamente utilizada no mundo, sendo essencial para o desenvolvimento web. Ele cofundou o projeto Mozilla, que deu origem ao navegador Firefox, e posteriormente fundou a Brave Software. A ubiquidade do JavaScript o torna uma das invenções mais importantes da história da computação.
