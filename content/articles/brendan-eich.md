## Summary

Brendan Eich is an American technologist who created the JavaScript programming language in 1995 while working at Netscape. JavaScript became the most widely used programming language in the world, essential for web development. He co-founded the Mozilla project, which led to the Firefox browser, and later founded Brave Software. JavaScript's ubiquity makes it one of the most important inventions in computing history.
